ふりかえりタブ（S03）

週/月/年のリズムカード・道具箱・これまでの軌跡を集約するアプリ中核タブ。

実装済 画面ID: LR-S03

目的・役割

「ふりかえり」タブはこのアプリの中核（★メイン機能）。日々の記録を素材として、 週・月・年の節目に「立ち止まり、言葉にする」振り返りセッションへの入口を提供する。 また道具箱（カレンダー・統計）でデータを俯瞰する機能と、 過去の振り返りを蓄積した「これまでの軌跡」タイムラインへの導線も集約する。

C案規律（グロー禁止・金は線でのみ）: 金面は使わない。リズムカードの isDue 状態のみ goldDim 枠線（1px）＋見出し明度上げで静かに格上げする。グロー・星・絵文字は出さない。

UI 構成（ReflectHomeView）

要素種類説明
NavigationStackコンテナタイトル「ふりかえり」（明朝インライン）。
ヒーローブロック（hero）VStack テキスト「すこし、立ち止まって\nみませんか」（明朝 26pt・moonWhite・lineSpacing 9）＋サブテキスト（14pt・textMuted）。
リズムカード群（rhythmCards）VStack（spacing 12）週次・月次・年次の 3 枚の RhythmCard。各カードにタップで振り返りセッション（fullScreenCover）を起動。
区切り線Rectangle（cardBorder 1px）リズムカードと道具箱の間の仕切り。
道具箱セクション（toolbox）VStack + NavigationLinkセクション見出し「道具箱」。カレンダー（LR-TOOLBOX-CAL）・統計（LR-TOOLBOX-STATS）への ToolCard 導線。
これまでの軌跡（trajectory）VStack + NavigationLinkセクション見出し「これまでの軌跡」。振り返り履歴が空なら EmptyStateView（moon.stars・「まだ星のない夜」）。履歴ありなら TrajectoryPreview（最新1件プレビュー）→ ReflectionTimelineView へ。

RhythmCard（リズムカード）の詳細

要素説明
SF Symbol アイコン週次: calendar / 月次: moon / 年次: moon.stars（26pt・isDue で amber・それ以外 textMuted）
見出しテキスト「週の振り返り」「月の振り返り」「年の振り返り」（16pt semibold・isDue で onSurface・それ以外 onSurfaceVariant）
「そろそろ」バッジisDue が true のとき表示。amber テキスト + goldDim capsule 枠線。
メタテキスト前回からの日数（「前回から ○日」）または初回文言（12.5pt・textMuted）
「今、ふりかえる」サブテキストisDue が true のとき表示（12pt・amber 80%透過）
カード枠線isDue: goldDim 1px / 非 isDue: cardBorder 1px。カード padding: isDue 20px / 非 isDue 16px
chevron.right右端に常時表示（outline 色）

isDue（そろそろ促進）判定ロジック

種別判定条件
週次（weekly）前回の completedAt から 7 日以上経過（または未実施）
月次（monthly）前回の completedAt が今月ではない（または未実施）
年次（yearly）前回の completedAt が今年ではない（または未実施）

サブ画面（ふりかえりタブから遷移）

振り返りセッション（ReflectionSessionView）— fullScreenCover

5 ステップフロー + 完了画面。詳細は LR-SESSION-01 および 機能仕様書 を参照。

STEPサブ画面内容
STEP 1ReflectionStepReview対象期間のログ一覧を表示（期間ラベル付き）。「次へ」で STEP 2 へ。
STEP 2ReflectionStepChoose「心に残った記録」を複数選択（isHighlight フラグ対象）。
STEP 3ReflectionStepPrompts「静かな問い」3 問への自由記述回答。
STEP 4ReflectionStepMood期間の総括感情（5 種）を選択。
STEP 5ReflectionStepNote気づきの自由記述（任意）＋「完了」ボタンで保存。
完了画面ReflectionCompletionView星がひとつ灯る（唯一許可するグロー）。「軌跡を見る」「閉じる」ボタン。
中断アラート: 入力あり（ハイライト選択・回答・感情・ノートいずれか）の状態で閉じようとすると「振り返りを中断しますか？」アラートを表示。「中断する」でデータは破棄（保存なし）。

これまでの軌跡（ReflectionTimelineView）

振り返り履歴を蓄積した星図タイムライン。詳細は LR-TIMELINE-01 を参照。

カレンダー（CalendarView）— 道具箱

記録のヒートマップ・日付選択でログ展開。詳細は LR-TOOLBOX-CAL を参照。

統計（StatsView）— 道具箱

サマリー 4 カード・カテゴリ円グラフ・感情の移ろいグラフ。詳細は LR-TOOLBOX-STATS を参照。

状態

状態表示内容
初期表示（振り返り未実施）全リズムカードが isDue=true 状態（「そろそろ」バッジ + goldDim 枠）。「これまでの軌跡」は EmptyStateView（「まだ星のない夜」）。
週次のみ実施済み週次カードは isDue に応じて通常または促進状態。月次・年次は isDue=true。
セッション中fullScreenCover でオーバーレイ表示。背後のタブは維持。interactiveDismissDisabled(true) で誤スワイプを防止。
軌跡あり「これまでの軌跡」に TrajectoryPreview（最新の note または最初の answer）が表示される。

遷移

操作遷移先
リズムカード（週次）タップReflectionSessionView（type: .weekly）を fullScreenCover で表示
リズムカード（月次）タップReflectionSessionView（type: .monthly）を fullScreenCover で表示
リズムカード（年次）タップReflectionSessionView（type: .yearly）を fullScreenCover で表示
道具箱「カレンダー」タップCalendarView へ NavigationLink push
道具箱「統計」タップStatsView へ NavigationLink push
「軌跡を見る」（TrajectoryPreview）タップReflectionTimelineView へ NavigationLink push
セッション完了画面「軌跡を見る」タップセッションを dismiss → onViewTimeline コールバック → navigateToTimeline = true → ReflectionTimelineView へ push

使用するデータモデル

// 振り返り履歴（completedAt 降順）
@Query(sort: \ReflectionSession.completedAt, order: .reverse)
private var history: [ReflectionSession]

// セッション起動管理
@State private var sessionType: ReflectionType?   // nil = 表示なし

// ReflectionSession（SwiftData @Model）
//   var typeRaw: String           // ReflectionType.rawValue
//   var periodStart: Date
//   var periodEnd: Date
//   var moodSummaryRaw: String?   // LogMood.rawValue
//   var answers: [ReflectionAnswer]
//   var highlightLogIds: [UUID]
//   var note: String?
//   var completedAt: Date

// isDue 判定（例: 週次）
//   Calendar.current.dateComponents([.day], from: last.completedAt.startOfDay, to: now).day >= 7

アクセシビリティ対応

  • リズムカードはボタンとして実装（Button + .buttonStyle(.plain)）。VoiceOver は「週の振り返り、ボタン」等で読み上げ。
  • 道具箱 ToolCard は NavigationLink（ボタン扱い）。chevron.right は装飾のためアクセシビリティ非対象。
  • セクション見出し（SectionHeader）には accessibilityAddTraits(.isHeader) を付与（SectionHeader 共通コンポーネント側で設定）。
  • ReflectionSessionView 内の ProgressBar に accessibilityLabel / accessibilityValue を設定（進捗 3/5 等）。

実装メモ

  • ReflectHomeView は NavigationStack を持つ。ReflectionSessionView のみ fullScreenCover（他は NavigationLink push）。
  • セッション完了後の「軌跡を見る」は dismiss + コールバックのパターン（ネストした NavigationStack を避けるため）。onViewTimeline: (() -> Void)? に渡す。
  • navigateToTimeline の navigationDestination(isPresented:) で ReflectionTimelineView へ push する。
  • isDue 判定と meta テキストは ReflectHomeView 内の private func で計算。history（@Query）から latest(_ type:) で最新セッションを取得する。
  • グロー・星エフェクトはセッション完了画面（ReflectionCompletionView）にのみ許可。ReflectHomeView 本体には一切出さない。

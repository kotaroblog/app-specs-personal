これまでの軌跡（星図タイムライン）

振り返り数だけ星を灯す星図・タリー（累計/連続週/ハイライト計）・縦タイムライン。

S2実装済 画面ID: LR-TIMELINE-01

目的・役割

「蓄積された振り返りそのものがアプリ固有の価値資産」（DESIGN.md 4.4）を体現する画面。 過去の振り返りを時系列カードで積み上げ、読み返せる「振り返りの軌跡」として提示する。 「星＝あなたの振り返り」と再定義し、C案「ダーク天体」の最大の差別化要素として星図を上部に配置。 壁紙としての星は全廃し、振り返り1回ごとに意味のある1粒が灯る。

UI 構成

要素種類説明
StarMapViewCanvas描画振り返りセッション数だけ小星を描画。シードRNG（seed: 20260613）でクラスタ配置。グロー禁止・textBody 色の2px前後。
キャプションText（明朝11.5pt）「あなたが振り返った夜の数だけ、星が灯る」（textMuted）
タリーカードAppCard（3列）これまでの振り返り（累計）/ 連続した週 / 残したハイライト。等幅24pt数字 + 11pt補助ラベル。
縦タイムラインForEach + ReflectionTilecompletedAt 降順で全件表示。各タイルに種別バッジ・期間・総括感情・気づきメモ冒頭・ハイライト数を表示。
空状態（EmptyNight）EmptyStateView風「まだ星のない夜」明朝16pt + 説明文。「最初の振り返りを始める」CTAボタン（金ヘアライン枠 + 明朝15pt）。

StarMapView 描画仕様

項目仕様
星の数振り返りセッション数（= starCount）
座標生成シード固定（20260613）の擬似乱数 SeededGenerator。再描画でも位置がぶれない。
クラスタ配置clusterCount = ceil(starCount / 4)。クラスタ中心から散らすことでノイズ感を回避。
星のサイズradius: 1.0〜1.25px（均等サイズ禁止。3段階相当のゆらぎ）
星の明度opacity: 0.5〜0.75（均等明度禁止）
色AppColors.textBody（#C9C5BA）。紫・青グラデ禁止。
グロー禁止（外周 box-shadow 相当は一切使わない）
高さframe height: 150px。RoundedRectangle(cornerRadius: 16) + cardBorder 1px 枠。

タリー集計ロジック（TimelineTally）

  • total: sessions.count（全種別の合計）
  • highlightCount: sessions の highlightLogIds.count を reduce 集計。
  • streakWeeks: weekly セッションの periodStart を週月曜 0:00 に正規化 → 降順でひと続きになっている週数をカウント。1週ずつ途切れた時点で終了。

状態

状態表示内容
振り返り0件EmptyNight（空状態）。「最初の振り返りを始める」CTA → fullScreenCover で weekly セッション起動。
振り返り1件以上StarMapView + キャプション + タリーカード + 縦タイムライン。

遷移

操作遷移先

使用するデータモデル

@Query(sort: \ReflectionSession.completedAt, order: .reverse)
private var sessions: [ReflectionSession]

// StarMapView に starCount として sessions.count を渡す
StarMapView(starCount: sessions.count)

// タリー集計
let tally = TimelineTally(sessions: sessions)

実装メモ

  • StarMapView は Canvas API（SwiftUI）で描画。CPU 負荷はほぼゼロ（静的描画）。
  • SeededGenerator はカスタム RNG（RandomNumberGenerator 準拠）。seed: 20260613 固定。
  • 空状態のCTAボタンスタイル: 金ヘアライン枠（goldDim 1px）、明朝テキスト、amber 色。グラデ・グロー禁止。
  • 振り返り詳細画面（ReflectionDetailView）は S3 以降の実装予定。現状は ReflectionTile は非タップ。
  • 年次振り返りの特別演出（その年のハイライト写真まとめ）は S4 以降で検討。

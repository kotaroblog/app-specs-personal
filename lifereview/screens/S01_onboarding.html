オンボーディング（S01）

初回起動時に生年月日・目標年齢を入力し UserProfile を作成する。C案「ダーク天体」トーン全面適用。

実装済 画面ID: LR-S01

目的・役割

アプリを初めて起動したユーザーに「人生を振り返るアプリ」であることを静かに伝え、 残り時間・週グリッドのパーソナライズに必要な生年月日と目標年齢を収集する。 C案「ダーク天体」トーン（墨と藍・琥珀の真鍮・明朝×ゴシック二層）を最初から体験させ、 アプリのムードを印象づける入口となる。

表示トリガー: @AppStorage("onboardingCompleted") が false（未設定含む）の場合、 起動直後に .fullScreenCover で表示する。「はじめる」ボタン経由で true に設定してから閉じる。 スワイプダウンでの dismiss は interactiveDismissDisabled(true) で禁止。

状態

状態表示・挙動
初期表示生年月日の初期値は 1995-01-01（DatePicker のデフォルト）。目標年齢は 80 歳。CTA は常に活性。
入力中DatePicker のホイール操作で日付が更新される。Stepper 操作で目標年齢が 40〜120 の範囲でインクリメント／デクリメント。
保存成功UserProfile を modelContext に insert + save → isCompleted = true → fullScreenCover が閉じてメインタブへ遷移。
保存失敗modelContext.rollback() 後に保存エラーアラートを表示。OK タップで再試行可能。

遷移

操作遷移先
「はじめる」ボタンタップ（保存成功）fullScreenCover を dismiss → RootTabView（今日タブ）へ
スワイプダウンinteractiveDismissDisabled(true) により無効化

使用するデータモデル

// SwiftData（書き込みのみ・読み込みは行わない）
@Environment(\.modelContext) private var modelContext

// ローカル入力ステート
@State private var birthDate: Date = ...   // 初期値: 1995-01-01
@State private var targetAge: Int = 80

// 保存アクション
func start() {
    let profile = UserProfile(birthDate: birthDate, targetAge: targetAge)
    modelContext.insert(profile)
    do {
        try modelContext.save()
        isCompleted = true    // @Binding → RootTabView の onboardingCompleted を true に
    } catch {
        modelContext.rollback()
        saveError = "プロフィールを保存できませんでした。もう一度お試しください。"
    }
}

// UserProfile（SwiftData @Model）
//   var birthDate: Date = Date()
//   var targetAge: Int = 80

アクセシビリティ対応

  • DatePicker に .accessibilityLabel("生年月日") を付与。
  • Stepper に .accessibilityLabel("目標年齢") + .accessibilityValue("\(targetAge)歳") を付与。
  • はじめるボタンはシステムフォントサイズに追従（minimumScaleFactor は未設定だが短文のため問題なし）。
  • 背景色は night1 固定（ダーク専用アプリのため preferredColorScheme の考慮不要）。

実装メモ

  • バックグラウンドは AppColors.background.ignoresSafeArea() で塗り、Launch 画面（night1）と同色にする。
  • ScrollView + .safeAreaInset(edge: .bottom) でコンテンツをスクロールさせつつ、「はじめる」ボタンを常に画面下部に固定する。
  • DatePicker の tint は amber 色。.labelsHidden() でシステムラベルを非表示にする。
  • 目標年齢は Stepper（40〜120）。値は amber 色 16pt semibold で左隣に表示し、視覚的に強調する。
  • 保存エラー時は modelContext.rollback() を必ず呼び出す。insert をキャンセルしないと再試行時に重複が発生する。
  • UserProfile は常に1件のみ存在する想定。重複防止は RootTabView 側でオンボーディングを once のみ表示することで担保する。

人生タブ（S04）

長期視点の集約画面。残り時間カウントダウン・人生の週グリッド・大切な人・バケットリスト・設定を提供する。

実装済 画面ID: LR-S04

目的・役割

「人生」タブは有限性を可視化し、長期的な視点でのアクション（バケットリスト・大切な人）への入口を提供する。 Your Life in Weeks の週グリッドと残り時間カウントダウンで、ユーザーが自分の人生の全体像を静かに俯瞰できる空間。 設定（歯車アイコン）は右上から開き、プロフィール編集・リマインダー設定・データ削除を集約する。

C案規律: 週グリッドの経過セル（amber 45%透過）と今週セル（red）のみが画面のアクセント。カウントダウン数字は onSurface。金面は使わない。

CountdownView の詳細

要素説明
残り時間表示profile.remainingYearsMonths() から年数・ヶ月数を取得。40pt semibold monospacedDigit。
ThinProgressBarprofile.progress()（0.0〜1.0）を amber 細線で表示。
進捗テキスト「これまで ○% を歩いてきました」（○ = Int(progress * 100)）。
アクセシビリティ.accessibilityElement(children: .combine)・accessibilityLabel「これからの時間」・accessibilityValue「○年○ヶ月。これまで○パーセントを歩いてきました」。

WeekGridBody / WeekGridCanvas の詳細

要素説明
サマリーカード現在年齢・経過週（amber）・残り週（onSurfaceVariant）・合計週（textMuted）の4カラム統計を AppCard 内に表示。縦セパレーターで区切る。
凡例「経過した週」（amber 45%透過）・「今週」（red）・「残りの週」（cardBorder）の3種。
WeekGridCanvas52列 × targetAge 行の週グリッドを Canvas で描画。cellW=(width-gutter-gap*52)/52・cellH=5px・rowGap=2px。10年毎に年齢ラベル（8pt・textMuted）を左端に表示。経過セル: amber 45%透過・今週セル: red・残りセル: cardBorder。
アクセシビリティCanvas 全体に「人生の週グリッド」accessibilityLabel・「経過○週、残り○週」accessibilityValue。

サブ画面（人生タブから遷移）

バケットリスト（BucketListView）

要素説明
List + BucketRowBucketItem を createdAt 降順で一覧表示。各行: チェックボタン（checkmark.circle.fill / circle）＋タイトル＋メモ（任意）＋目標期日（任意）。
完了状態isCompleted=true のとき取り消し線＋textMuted 色＋チェックアイコン amber 色。
スワイプ削除.onDelete で削除。確認アラートなし（List の標準動作）。
EmptyStateView未登録時: checklist アイコン＋「まだ何もありません」＋「やりたいことを追加」CTA ボタン。
追加（BucketEditSheet）ナビバー「+」またはEmptyState CTA タップで .sheet 表示。タイトル（必須）・メモ（任意）・目標期日（Toggle + DatePicker）を入力して保存。

大切な人（RelationsListView）

要素説明
List + RelationRowRelation を createdAt 降順で一覧表示。各行: 名前（subheadline semibold）＋続柄（caption・任意）＋残り会える回数「あと○回」（amber・16pt semibold）。
残り回数計算currentAge が設定されている場合: (targetAge - currentAge) × meetingsPerYear。設定なしの場合は「年○回」のみ表示。
スワイプ削除.onDelete で削除。
EmptyStateView未登録時: person.2 アイコン＋「まだ登録がありません」＋「大切な人を追加」CTA ボタン。
追加（RelationEditSheet）ナビバー「+」または EmptyState CTA タップで .sheet 表示。名前（必須）・続柄（任意）・年齢（Toggle + Stepper）・会う頻度（Stepper・年1〜365回）を入力して保存。

セクション項目説明
プロフィール目標年齢タップで TargetAgeSheet（Stepper 40〜120・amber 44pt 数字）を .sheet 表示。保存で UserProfile.targetAge を更新。
プロフィール生年月日タップで BirthDateSheet（DatePicker graphical スタイル）を .sheet 表示。保存で UserProfile.birthDate を更新。
通知リマインダーNavigationLink → ReminderSettingsView（LR-REMINDER-01）。現在の設定サマリー（「オフ」または「日 21:00」形式）を右に表示。
アプリ情報バージョン「1.0.0」固定テキスト。
アプリ情報プライバシーポリシーLink → SFSafariViewController で外部 URL を開く。
アプリ情報利用規約Link → 外部 URL。
アプリ情報サポート・お問い合わせLink → 外部 URL。
危険操作全データを削除destructive ロールのボタン。確認アラート（「すべての記録・振り返り・設定が削除されます」）経由で実行。削除後に onboardingCompleted = false でオンボーディングに戻る。

@Query private var profiles: [UserProfile]

// UserProfile（SwiftData @Model）
//   var birthDate: Date
//   var targetAge: Int
//   func remainingYearsMonths() -> (years: Int, months: Int)
//   func progress() -> Double
//   func elapsedWeeks(from:) -> Int
//   func currentAge(from:) -> Int

// BucketListView
@Query(sort: \BucketItem.createdAt, order: .reverse) private var items: [BucketItem]
// BucketItem（SwiftData @Model）
//   var title: String
//   var memo: String?

//   var targetDate: Date?

// RelationsListView
@Query(sort: \Relation.createdAt, order: .reverse) private var relations: [Relation]

アクセシビリティ対応

  • CountdownView: .accessibilityElement(children: .combine) + accessibilityLabel/Value でまとめて読み上げ。
  • WeekGridCanvas: .accessibilityLabel("人生の週グリッド") + .accessibilityValue("経過○週、残り○週")。
  • 設定ナビバー「gearshape」ボタン: .accessibilityLabel("設定を開く")。
  • RelationsListView「+」ボタン: .accessibilityLabel("大切な人を追加")。
  • BucketListView「+」ボタン: .accessibilityLabel("やりたいことを追加")。
  • BucketRow のチェックボタン: .accessibilityLabel(item.title) + .accessibilityValue(isCompleted ? "完了済み" : "未完了") + .accessibilityAddTraits(isCompleted ? .isSelected : [])。
  • WeekGridBody サマリーの各 stat セル: .accessibilityElement(children: .combine) でまとめる。

実装メモ

  • LifeView は NavigationStack を持つ。SettingsView・RelationsListView・BucketListView は全て同一 NavigationStack 内に push される。
  • WeekGridCanvas は Canvas（SwiftUI）で描画するため非常に軽量。targetAge=80 では 80×52=4,160 セル分の描画をこなす。
  • WeekGridBody の totalWeeks / elapsedWeeks / currentAge は UserProfile の extension（UserProfile+Life.swift）で計算。ビューに計算ロジックは持たせない。
  • BucketItem・Relation の削除は SwiftData の modelContext.delete() → save()。失敗時は rollback + エラーアラート。
  • SettingsView の「全データを削除」は try modelContext.delete(model: LogEntry.self) 等を種別毎に実行し、最後に save()。失敗時は rollback。
  • SettingsView のプロフィール編集シート（TargetAgeSheet / BirthDateSheet）は UserProfile を直接更新し save() する。
  • 残り回数計算（RelationRow）は currentAge が nil の場合に nil を返し、View 側で分岐する。計算は純粋な算術: (targetAge - currentAge) × meetingsPerYear（0以下は max(0,…)でクランプ）。

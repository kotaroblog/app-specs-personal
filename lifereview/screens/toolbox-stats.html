道具箱: 統計

サマリー4カード・カテゴリ円グラフ・カテゴリ内訳・感情の移ろいグラフ。

S3実装済 画面ID: LR-TOOLBOX-STATS

目的・役割

日々の記録（LogEntry）を集計し、カテゴリ分布・記録頻度・感情の移ろいをビジュアルで把握する画面。 振り返りの「道具箱」として「自分はどんなことをよく記録しているか」「感情がどう変化しているか」を俯瞰する用途。 Phase S3 で感情の移ろいグラフ（Swift Charts 月別スタック棒グラフ）を追加実装済み。

UI 構成

要素種類説明
サマリーカード（4枚）LazyVGrid 2列 × SummaryCard総記録数 / 今月の記録日数 / 最多カテゴリ / 記録率。各カード: SF Symbol + 大きな数値（22pt bold）+ ラベル（12pt textMuted）。
カテゴリ別割合（円グラフ）Swift Charts SectorMarkドーナツ型。各カテゴリの件数割合をカテゴリカラーで表示。innerRadius: 0.5。angularInset: 1.5。cornerRadius: 3。
カテゴリ内訳リストForEach + HStack件数降順。アイコン + カテゴリ名 + 件数 + 割合（%）。
空状態EmptyStateViewログ0件時: 「記録をつけると統計が表示されます」。chart.bar アイコン（64px）。

集計ロジック

指標計算方法
総記録数logs.count
今月の記録日数今月のログを year × month でフィルタ → 日（day）の Set のカウント
最多カテゴリbreakdown（件数降順）の first?.category
記録率今月記録日数 / 今月経過日数 × 100（%）。小数1桁表示。
カテゴリ内訳LogCategory.allCases × 件数 Dict。0件は非表示。件数降順ソート。

感情の移ろいグラフ（S3 追加）

項目仕様
グラフ種別Swift Charts BarMark（スタック棒グラフ）
表示期間直近6か月（今月を含む6か月分）
X軸月（YYYY-MM 形式を "M月" に整形して表示）
Y軸件数（ラベル自動、目盛り4本程度）
スタック方式LogMood 5種をそれぞれ積み上げ。感情ごとに foregroundStyle で色分け。
感情カラーjoy: #EAB308 / fulfilled: #2563EB / calm: #0F766E / tired: #64748B / anxious: #9333EA（LogMood カラーと同一）
凡例グラフ下部にインライン凡例（感情ラベル + 色サンプル丸。横スクロール可）
空状態感情タグ付き記録が0件の場合: 「感情タグをつけて記録すると、感情の移ろいが表示されます」をグラフエリア中央に表示

感情グラフの集計ロジック

// 直近6か月の (year, month) ペアを生成
let months: [(year: Int, month: Int)] = ...  // 降順で6か月

// moodData: [(month: String, mood: LogMood, count: Int)]
// logs から moodRaw != nil のものを抽出 → (year, month, mood) でグループ集計
let moodData = logs
    .compactMap { log -> (Date, LogMood)? in
        guard let raw = log.moodRaw,
              let mood = LogMood(rawValue: raw) else { return nil }
        return (log.date, mood)
    }
    .filter { /* 直近6か月 */ }
    .grouped(by: ...) // (monthLabel, mood) → count

状態

状態表示内容
ログ0件EmptyStateView（chart.bar / 64px / 「記録をつけると統計が表示されます」）。サマリーカード・グラフは非表示。
ログあり・感情タグなしサマリー → 円グラフ → カテゴリ内訳 → 感情グラフ（空状態メッセージ）の順で表示。
ログあり・感情タグありサマリー → 円グラフ → カテゴリ内訳 → 感情の移ろいグラフ（スタック棒）を縦スクロールで表示。

実装メモ

  • Swift Charts（import Charts）を使用。iOS 16+ 必須。本アプリは iOS 18+ なので問題なし。
  • SummaryCard は private struct として StatsView 内に定義。AppCard（cornerRadius: 12）ベース。値テキストは .minimumScaleFactor(0.6) で長い文字列も収める。
  • breakdown は computed var で毎回計算（ログ数が少ないため許容）。ログ数が増える場合はメモ化を検討。
  • カテゴリカラーは LogCategory.color（Flutter 版の配色を踏襲）。
  • 感情グラフの BarMark は .foregroundStyle(by: .value("感情", mood.label)) + .chartForegroundStyleScale でカラーマッピングを明示指定する。
  • 6か月分データが存在しない月（記録0件）はスタック棒を表示しない（X軸のラベルは表示）。

ふりかえりホーム

振り返りタブの入口。週/月/年のリズムカード・道具箱・これまでの軌跡を集約する。

S2実装済 画面ID: LR-REFLECT-HOME

目的・役割

ユーザーが「振り返り」タブを開いたときの起点画面。 週/月/年それぞれのリズムカードで「立ち止まる」タイミングを静かに促し、道具箱（カレンダー/統計）への導線と、 過去の軌跡プレビューを1画面に集約する。 C案「ダーク天体」規律に従い、グロー禁止・金は線のみ・1画面1灯（週次カード見出しに明度を集約）。

状態

状態表示内容
初回起動（振り返り0件）軌跡セクションに EmptyState「まだ星のない夜」。リズムカードの meta は「はじめての振り返り」等の固定文言。
振り返り履歴あり軌跡セクションに TrajectoryPreview（最新1件のプレビューテキスト + 「軌跡を見る」）。リズムカードの meta に「前回からN日」を表示。
週次カード（isDue: 自動判定）前回週次振り返りから7日以上経過（or 週次振り返り未実施）の場合 isDue: true → 金ヘアライン罫・「そろそろ」バッジ表示。
月次カード（isDue: 自動判定）今月の月次振り返りが未実施の場合 isDue: true。
年次カード（isDue: 自動判定）今年の年次振り返りが未実施の場合 isDue: true。

isDue 判定ロジック（S3 実装済）

種別isDue = true の条件
週次（weekly）週次の振り返りが一度もない または 最後の週次振り返りの periodStart から現在まで 7日以上経過
月次（monthly）今月（year + month）の月次振り返りが0件
年次（yearly）今年（year）の年次振り返りが0件
// ReflectHomeViewModel の isDue 計算例
func isDue(for type: ReflectionType, history: [ReflectionSession]) -> Bool {
    let sessions = history.filter { $0.typeRaw == type.rawValue }
    switch type {
    case .weekly:
        guard let last = sessions.first else { return true }
        return Calendar.current.dateComponents([.day], from: last.periodStart, to: .now).day ?? 0 >= 7
    case .monthly:
        return !sessions.contains { Calendar.current.isDate($0.periodStart, equalTo: .now, toGranularity: .month) }
    case .yearly:
        return !sessions.contains { Calendar.current.isDate($0.periodStart, equalTo: .now, toGranularity: .year) }
    }
}

実装メモ

  • sessionType: ReflectionType? を @State で持ち、.fullScreenCover(item: $sessionType) でセッション画面を表示する（Identifiable 準拠により種別ごとに自動生成）。
  • RhythmCard の isDue は S3 で自動判定に切り替え済み（固定値から上記ロジックへ変更）。
  • TrajectoryPreview のプレビューテキスト優先順: note（気づきメモ）→ answers[0].answer → nil（fallback: 「直近の振り返りが残っています」）。
  • C案規律: 金（AppColors.amber / goldDim）は isDue カードのヘアライン・バッジ・TrajectoryPreview の引用符のみ。ホーム・セッション中は星を出さない（完了画面でのみ灯す）。
  • ヒーローテキストは AppFont.mincho(26)（Shippori Mincho 相当）。

ゲームメカニクス

コアルール

  • 4×4 グリッド固定
  • 2048 標準合体（同種2体 → 次段階に進化、1手で1タイル1回まで）
  • シズクのみがランダム spawn（初期2個 + 毎手1個）
  • 手数制限 + Goal 達成でクリア
  • 無料リトライ無制限 + 完全リスタート
  • 失敗条件: 手数切れ ∨ 盤面詰み（合体可能ペアなし）
  • Stage 24 クリア後: 無限モード（森を歩く）へ移行

※ collectedCounts は「合体で生まれた結果のみ」をカウント。spawn は対象外。シズクは Goal にできない（合体結果として生まれないため）。

触感・演出

タイル単体

  • 個別呼吸 — 精霊ごとに duration / amplitude が異なる、位相もランダム
    • シズク: 中庸 / ソラ: 軽く速め / ハナ: 柔らかい / ルナ: 最遅 / ホシ: 短周期 / ソラノ: 重力感 / ユメ: 最遅最小 / ニジ: オーロラと別軸で揺らぐ
  • 柔らかい影 — CIGaussianBlur, alpha 0.18, y -4
  • 月光ハロー — 半透明グロー + 4.5〜6.5 秒の alpha 脈動
  • 共鳴ハロー — 同種隣接時に薄く脈動（無言の合体誘導）
  • swipe 反応 — touchesBegan で全タイル scale 0.99、touchesEnded で復帰

特殊レンダリング

  • ニジタイル — body・輪郭・絵文字・影を全削除。3層の青紫オーロラ（gradient sprite）が drift + alpha shimmer で揺らぐ。「物体」ではなく「現象」
  • ルナタイル — 月光ミスト（外側の極薄 gradient sprite）追加

合体3段階演出

フェーズ時間内容
Phase 1（引き寄せ）0.08s2タイルが target へ加速移動 + scale 1.0→0.75 + alpha 1.0→0.85
Phase 2（融合光）0.06sHaptics 同フレーム発火（rainbow 到達なら .evolve、それ以外は .merge）
Phase 3（誕生）0.14s新タイルが光が薄れる中で fade-in + scale 0.5→1.06→1.0

環境音（プロシージャル生成・外部音源ゼロ）

  • 夜風 — 強ローパスフィルタ + 12 秒ループ + 0.15Hz 振幅変調
  • 葉ずれ — 白色雑音の差分（簡易ハイパス）を低レベルで重ねる
  • 微鈴 — 620/740/880/1040/1180Hz からランダム選択、指数減衰、18〜32秒間隔
  • 音量ピーク 0.08（「無音」と感じる境目）
  • .ambient + .mixWithOthers で他アプリ音楽を遮らない
  • scenePhase 連動で background 時停止

Haptics

  • swipe — Soft impact（intensity 0.55）
  • merge — Medium + CHHapticEngine の余韻パターン
  • evolve（虹到達） — Heavy + 長め残響パターン

盤面全体

  • 呼吸 — scale 1.000 ↔ 1.012 / 4秒
  • 微細揺れ — x±0.4pt / y±0.6pt、X と Y は独立 7〜12 秒周期（Lissajous 軌道）
  • 大気パーティクル — 青白い粒子が画面下から上へ漂う（SKEmitterNode）
  • 遠景ミスト — 薄い青の矩形シャドウ
  • 盤面背景 — 半透明角丸ボックス + 16 セル枠線

設計判断メモ

  1. calm路線 → ステージ制転換 — 当初 "calm/ambient/tactile" 路線で MVP 構築。「虹後の展開がない」「永続的に楽しめない」を実プレイ中に懸念。calm 路線を触感レイヤーとして残しつつ、ステージ制 + キャラ × 8 体 × 進化で「次の存在に会いたい」を駆動軸に転換。
  2. 触感の核 — 「動きでキャラ性を出す」「演出より呼吸」「現象 vs 物体」が判断軸。AnimationTiming 単一ファイルに全時間定数を集約し、易調整。
  3. 絵文字 → キャラ画像移行 — TileNode に loadTexture(for:) フォールバックを実装。画像未読込時は絵文字へ自動回帰。

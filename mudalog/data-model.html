データモデル

SwiftData @Model の構造とリレーション

ER 図

erDiagram WasteLog { UUID id PK String title Date createdAt String memo Bool isArchived String interestLevelRaw String feelingRaw String tagsRaw } FocusPlan { UUID id PK String themeTitle Date createdAt Bool isActive Date completedAt } FocusPlanItem { UUID id PK Int day String title String statusRawValue Date createdAt Date completedAt } FocusPlan ||--o{ FocusPlanItem : "cascade / inverse"
WasteLog と FocusPlan は直接リレーションを持たない。テーマ（themeTitle）の文字列マッチングによって MudaInsightEngine が関連付けを行う。

WasteLog

ユーザーが記録する「ムダ行動」1件を表すメインエンティティ。

@Model
final class WasteLog {
    @Attribute(.unique) var id: UUID = UUID()
    var title: String = ""
    var createdAt: Date = Date.now
    var memo: String = ""
    var isArchived: Bool = false
    // 以下3フィールドはCSV文字列またはrawValueとして永続化し、
    // computedプロパティ経由で型変換する（SwiftDataマイグレーション対策）
    private var interestLevelRaw: String = InterestLevel.undecided.rawValue
    private var feelingRaw: String = ""
    private var tagsRaw: String = ""
}

InterestLevel（興味レベル）

rawValue表示文字列説明
まだわからないまだわからない初期値。判断保留
もういいもういい熱が冷めた
ちょっと気になるやりたいかも軽い関心
またやりたいまたやりたい積極的に再訪したい
なぜか忘れられない忘れられない最高熱中度

Feeling（そのときの気分）

rawValue説明
気楽リラックスして行った
夢中没頭した
衝動衝動的にやった
すぐ飽きたすぐ飽きた

tags（タグ）

tagsRaw にカンマ区切りCSVで保存。取得時は decodeTags() で [String] に変換。 空文字列フィルタリング済み。最大タグ数に上限なし（UX上は3〜5個を想定）。

FocusPlan

7日プランの単位。ユーザー1人につき同時に isActive = true のプランは1件のみ許可。

@Model
final class FocusPlan {
    @Attribute(.unique) var id: UUID = UUID()
    var themeTitle: String = ""
    var createdAt: Date = Date.now
    var isActive: Bool = true
    var completedAt: Date? = nil
    @Relationship(deleteRule: .cascade, inverse: \FocusPlanItem.plan)
    var items: [FocusPlanItem] = []
}

FocusPlanItem

7日プランの各日アクション（Day 1〜7）を表す。

@Model
final class FocusPlanItem {
    @Attribute(.unique) var id: UUID = UUID()
    var day: Int = 0
    var title: String = ""
    var statusRawValue: String = PlanItemStatus.notStarted.rawValue
    var createdAt: Date = Date.now
    var completedAt: Date? = nil
    var plan: FocusPlan? = nil
}

PlanItemStatus

rawValue意味
notStarted未着手
today今日のアクション（1件のみ存在）
completed完了済み

MudaInsightEngine（ロジック層）

SwiftData エンティティではなく純粋関数群。WasteLog 配列を入力として以下を算出する。

メソッド出力
themeInsights(from:)タグ出現頻度 → [ThemeInsight]
hotThemes(from:)上位3テーマ（2回以上登場）
recentThemeTitles(from:)直近7日のテーマ名一覧
branches(for:logs:limit:)テーマに関連するサブログタイトル群（伏線マップ用）

マイグレーション方針

SwiftDataの @Model は全プロパティにデフォルト値必須。新規プロパティ追加時は必ずデフォルト値を設定する。 interestLevelRaw / feelingRaw / tagsRaw は private String で永続化する設計を維持し、Enum型を直接 @Model に持たせない。

SwiftData @Model の構造とリレーション

ER 図

erDiagram EntityA ||--o{ EntityB : has EntityA { UUID id String name Date createdAt } EntityB { UUID id String value }

EntityA

@Model
final class EntityA {
    var id: UUID = UUID()
    var name: String = ""
    var createdAt: Date = Date()

    @Relationship(deleteRule: .cascade)
    var children: [EntityB] = []
}

マイグレーション方針

SwiftDataの@Modelは全プロパティにデフォルト値必須。新規プロパティ追加時は必ずデフォルト値を設定する。

ゲーム画面（GameView）

メイン画面（タイトル+ゲーム+結果が統合）

v1.0画面ID: CR-GAME-01

UI 構成

要素説明
タイトルバー「CHROMATIC」中央 + リーダーボード/設定ボタン両脇
ScoreHeaderView現スコア + 全難易度ベストスコア
StreakBadgeView連続プレイ日数（炎アイコン）
StatusTextView状態表示（「WATCH」「YOUR TURN」「GAME OVER」）
PanelGridView2×2パネル（赤/緑/青/黄）
SequenceDotsViewシーケンスドットインジケーター
ProgressBarView入力進捗
難易度Pickeridle時のみ表示（NORMAL/FAST/INSANE）
STARTボタンラベル動的（START / RETRY等）
BannerAdView下部（広告削除なし時）
MilestoneFlashViewラウンド5/10/15/20到達時の演出
GameOverOverlayViewゲームオーバー時のオーバーレイ

状態（GameState）

状態表示
idle難易度選択 + STARTボタン
showingシーケンス表示（点灯）
playerTurnパネルタップ受付
gameOverGameOverOverlayView 表示

遷移

操作遷移先
STARTタップshowing 状態へ
正解シーケンス満了で次ラウンド
不正解gameOver
RETRYInterstitial表示 → 新ゲーム
MODE SELECTInterstitial表示 → idle
SHAREShareCardView → ShareSheet
トロフィーアイコンGame Center Leaderboard
設定アイコンSettingsView (sheet)

実装メモ

  • scenePhase 監視 → refreshStreakOnLaunch()
  • ATT 要求は初回ゲームオーバー時
  • Interstitial は2ゲームに1回（gameCount % 2 == 0）
  • hasSeenOnboarding=false でOnboardingView (sheet, interactiveDismissDisabled)

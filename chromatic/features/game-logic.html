ゲームロジック

シーケンス記憶・速度加速・マイルストーン

v1.1機能ID: CR-FEAT-LOGIC

ゲームフロー

  1. startGame(): ストリーク記録 → sequence/playerIndex/score リセット → nextRound()
  2. nextRound(): ランダム色（0-3）追加 → 速度計算 → showSequence()
  3. showSequence(): 各色を順に点灯（litMs+音） → 消灯（gapMs） → playerTurn
  4. handlePanelTap(): 正解判定 → playerIndex+1
  5. シーケンス満了 → score+1 → checkMilestone() → nextRound()
  6. 不正解 → triggerWrongAnswerFeedback() → 約0.42秒後に triggerGameOver()

ミスフィードバック処理（wrongAnswerFeedback）

プレイヤーが誤ったパネルをタップした際、即ゲームオーバーにせず以下の順で処理する:

  1. state を wrongAnswerFeedback に変更（パネルタップを即時無効化）
  2. 正解パネル（現在の sequence[playerIndex] に対応するパネル）を約420ms間点灯
  3. StatusTextView に「正解はこのパネルでした」を表示
  4. 420ms 経過後、state を gameOver に変更 → GameOverOverlayView 表示
パラメータ値
点灯時間420ms（0.42秒）
フィードバック中のタップ受付無効（state != .playerTurn で自動ガード）

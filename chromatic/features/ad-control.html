InterstitialAdManager API

final class InterstitialAdManager {
    func loadAd()
    func showIfReady(completion: @escaping () -> Void)
    // 広告削除済 or 表示条件未達 → サイレントで完了
}

表示条件

  • 広告削除なし
  • 広告ロード済み
  • gameCount % 4 == 0（4ゲームに1回。v1.0〜v1.1 は 2ゲームに1回だったが、UX改善のため緩和）

ストリーク機能（StreakManager）

v1.1機能ID: CR-FEAT-STREAK

API

final class StreakManager {
    static let shared: StreakManager

    func recordPlay()  // ゲーム開始時
    func refreshStreakOnLaunch()  // 起動時 / scenePhase(.active)
}

UserDefaults

キー用途
chromatic_streak_count連続日数
chromatic_last_played_date最終プレイ日（YYYY-MM-DD）

処理フロー

  1. recordPlay(): 日付が変わっていれば前日プレイなら +1、なければ 1リセット
  2. refreshStreakOnLaunch(): 最終プレイから2日以上経過 → 0リセット

通知許可リクエスト

項目内容
要求タイミングストリーク2日以上達成時（streak_count >= 2 かつ未決定の場合のみ）
通知内容毎日21時のリマインダー（v1.1 ドーパミン強化アップデートから）
要求条件UNAuthorizationStatus が .notDetermined の場合のみリクエスト。.denied の場合は要求しない

UI 表示（StreakBadgeView）

日数色
0日非表示（opacity=0）
1-6日橙
7-29日金
30日+赤

CHROMATIC

4色2×2グリッドの色記憶パズル · Bundle ID: com.happyboy1002.CHROMATIC

概要

  • カテゴリゲーム / パズル
  • 対応OSiOS 18.0+ / Swift 6.0
  • 言語日本語のみ
  • マネタイズ広告（Banner/Interstitial）+ IAP（広告除去 ¥240）
  • ステータスリリース済
  • 配信地域日本のみ
  • 対応デバイスiPhone（縦向き専用）
  • 難易度NORMAL / FAST / INSANE
  • Game Center対応（難易度別リーダーボード3本）

アプリの目的

赤・緑・青・黄の4色パネルが順に光るシーケンスを記憶して同じ順にタップする色記憶パズル。 NORMAL/FAST/INSANEの3難易度、5/10/15/20ラウンドのマイルストーン演出、色覚サポート、Game Centerリーダーボード対応。

ゲームプレイ画面

SceneKit を使った 3D ボクセルワールドのメインプレイ画面。操作・HUD・各種モーダルを含む

画面ID: VG-GAME-01

目的・役割

ゲームの中核となる画面。SceneKit のレンダリングレイヤー（SCNView）の上に SwiftUI の HUD・操作ボタンレイヤーをオーバーレイする構成。サバイバルとクリエイティブの 2 モードを切り替えて遊べる。

UI 構成

操作コントロール

要素種類説明
左バーチャルジョイスティックDragGesture オーバーレイ左下隅に固定。ドラッグ量・方向をプレイヤーの水平移動速度に変換する
右エリアドラッグDragGesture（画面右半分）ドラッグ量を yaw（左右）・pitch（上下）視点回転に変換する
壊すボタンButton（長押し判定付き）長押し中、照準ブロックをツール採掘時間に応じて破壊する
置くボタンButtonタップで照準面に選択中ブロックを設置する
ジャンプボタンButton（長押し判定付き）タップでジャンプ。クリエイティブ時は長押しで上昇（フライ）
食べるボタンButton所持食料がある場合のみ表示。バッジで所持数を表示。タップで食料消費 → HP・空腹を回復

HUD（ヘッドアップディスプレイ）

要素位置説明
HP バー左上❤️ アイコン × 10。サバイバル時のみ表示。最大 20 HP を 2 HP / アイコンで表現
空腹バー左上（HP バー下）🍗 アイコン × 10。サバイバル時のみ表示。最大 20 を 2 / アイコンで表現
クリエイティブ表示左上クリエイティブモード時は HP・空腹バーの代わりに「∞」を表示
ホットバー下中央選択中ブロック種別アイコンと所持個数バッジを表示
ツールバー下部解放済みツール（素手・woodPickaxe・stonePickaxe・ironPickaxe・diamondPickaxe）を選択する行
天気切り替えボタン右上晴れ / 雨 / 雷雨をサイクル切り替え
クリエイティブ切り替えボタン右上（天気ボタン下）サバイバル / クリエイティブをトグル
照準（クロスヘア）中央画面中央に固定表示。照準先ブロックのハイライトと連動

シート / モーダル

シートトリガー内容
インベントリインベントリボタン全所持アイテムの種別・数量一覧。アイテムをタップでホットバーにセット
クラフト画面クラフトボタン解放可能なツールレシピ一覧。素材が揃っているレシピをタップでクラフト実行
かまどかまどブロック操作鉄鉱石 → 鉄インゴット、金鉱石 → 金インゴットに精錬。精錬中はプログレスバー表示
チェストチェストブロック操作チェストの中身一覧（スロット 27 個）。アイテムを預け入れ・取り出しする
設定画面設定ボタン感度・デッドゾーン調整スライダー、画質プリセット（標準 / 高画質）切り替え

状態

状態表示内容
サバイバルモードHP・空腹バーを表示。モブダメージ・落下ダメージ・空腹ダメージが有効
クリエイティブモード「∞」表示。フライ有効・ブロック破壊即時・所持数消費なし・全ダメージ無効
プレイヤー死亡画面が赤フェード → 「あなたは死にました」オーバーレイ → リスポーン後ワールド選択へ or スポーン地点に復帰
ブロック破壊中照準ブロックに亀裂エフェクト（採掘進行度に応じて 0〜5 段階）
インベントリ / シート表示中ゲームループを一時停止。操作コントロールを無効化

遷移

操作遷移先
戻るボタン / バックスワイプワールド選択画面へ pop。最終プレイ日時を更新してから遷移
各シートを閉じるゲームプレイ画面に戻る（ゲームループ再開）

実装メモ

  • SCNView を UIViewRepresentable でラップし、SwiftUI の ZStack 上に HUD をオーバーレイする構成をとる
  • ゲームループは SCNSceneRendererDelegate の renderer(_:updateAtTime:) で 60fps を目指す
  • ジョイスティック・右エリアドラッグは独立した DragGesture で処理し、multiTouch を有効化する
  • 食べるボタンは inventory に食料アイテムが 1 つ以上ある場合のみ表示する（.opacity 制御ではなく条件分岐で非表示）
  • シート表示中は SCNView のゲームループを isPaused = true で停止する
  • ホットバーのブロックアイコンは Assets.xcassets の UIImage テクスチャと一致させる
  • 設定画面の感度・デッドゾーン値は UserDefaults に保存し、GameInput 経由でジョイスティックロジックに反映する

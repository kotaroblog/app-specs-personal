採掘演出

ブロック破壊（採掘）に対する視覚フィードバック。採掘中のひび割れ表示と破壊完了時の破片パーティクル

機能ID: VG-FEAT-MINING

概要

プレイヤーが「壊す」ボタンを長押ししてブロックを採掘する際、進行度と破壊完了を直感的に伝えるための演出。採掘中は対象ブロックにひび割れが段階的に深まり、破壊した瞬間にブロック色の破片が飛び散る。これによりツールごとの採掘時間差や「あと少しで壊れる」という手応えを視覚的に提示する。採掘速度そのものの仕様はクラフト（採掘速度テーブル）を参照。

関連画面

  • ゲームプレイ画面（壊すボタン・照準）

処理フロー

sequenceDiagram participant U as User participant GV as GameView participant TA as TextureAtlas participant Scene as SceneKit Scene U->>GV: 壊すボタン長押し（照準ブロック） loop 採掘中（progress 0→1） GV->>GV: progress を更新 GV->>TA: crackStageImages[stage] を取得 GV->>Scene: breakOverlayNode を対象ブロックに追従表示 end Note over GV: progress ≥ 1.0 でブロック破壊 GV->>Scene: 対象ブロックノードを除去 GV->>GV: spawnBreakParticles(color: colorTop) GV->>Scene: SCNParticleSystem をワンショット再生 Note over Scene: 約0.55秒後に自動消滅

ひび割れ表示

項目仕様
表示ノード採掘中の対象ブロックに追従するオーバーレイノード（breakOverlayNode）。照準ハイライト（highlightNode）と同様の追従の仕組みで実装
テクスチャTextureAtlas.crackStageImages。決定論的ノイズパターンで生成した半透明の黒い線からなる 4 段階のひび割れテクスチャ
段階数4 段階（progress 0〜1 に対応）
段階の閾値progress 0.25 / 0.5 / 0.75 を境に次の段階へ切り替え
生成方式決定論的ノイズ（seed 固定）で生成するため、同一ブロックでは常に同じひび割れパターンになる

破片パーティクル

項目仕様
発生タイミングブロックが壊れた瞬間にワンショットで 1 回バースト（spawnBreakParticles）
実装SCNParticleSystem
破片の色壊したブロックの色（colorTop）を使用
飛散方向放射状（spreadingAngle 160 度）
運動重力の影響を受けて落下する
寿命約 0.55 秒後に自動消滅
対象全ブロック種別で共通発生（松明・植生・チェスト・農地・小麦・通常ブロック）

ビジネスルール

  • ひび割れは採掘進行度（progress: 0〜1）に連動し、progress 0.25 / 0.5 / 0.75 の閾値で 4 段階に切り替える
  • ひび割れテクスチャは決定論的ノイズで生成するため、生成結果はビルド間・プレイ間で一貫する
  • 破片パーティクルはブロック種別を問わず必ず発生し、破片色は破壊したブロックの colorTop を採用する
  • クリエイティブモードでのブロック破壊は即時のため、ひび割れ段階が視認されないケースがある（破片バーストは発生しうる）

外部連携

連携先用途備考
TextureAtlas（crackStageImages）ひび割れ 4 段階テクスチャの供給決定論的ノイズで生成
SceneKit（SCNParticleSystem）破片バーストの描画ワンショット・自動消滅
クラフト（VG-FEAT-CRAFTING）採掘進行度を決めるツール別採掘速度progress の進み方を規定

エラー処理

発生条件対応
採掘中に対象ブロックが照準から外れた / 採掘が中断されたbreakOverlayNode を除去し、progress をリセットする
crackStageImages の段階インデックスが範囲外閾値ロジックで 0〜3 にクランプし、範囲外参照を避ける

実装メモ

  • 実装箇所: GameView.swift（breakOverlayNode の追従表示、spawnBreakParticles のバースト生成）
  • 実装箇所: TextureAtlas.swift（crackStageImages のひび割れテクスチャ生成）
  • breakOverlayNode は highlightNode と同じく対象ブロックに毎フレーム追従させる
  • 段階切り替えは progress を 0.25 刻みの閾値で 4 段階にマッピングする
  • 破片パーティクルは spreadingAngle 160 度・重力あり・寿命約 0.55 秒でワンショット再生し、再生後は自動でノードを解放する

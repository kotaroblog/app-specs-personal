広告制御機能

Banner / Interstitial / Rewarded 3種広告

v1.0機能ID: PZ-FEAT-AD

広告構成

種類表示タイミング無料/購入
BannerGameView 下部に常時無料のみ
Interstitial3ゲームに1回、最低90秒間隔無料のみ
RewardedGameOver時の救済（1回限り）常時利用可

AdMob ID

App ID: ca-app-pub-4104853755894501~1889600698
Interstitial: 2184902139
Rewarded: 3655551463

AdManager API

@Observable @MainActor
final class AdManager {
    func loadInterstitial()
    func loadRewarded()
    func showInterstitialIfReady()  // 3ゲーム/90秒制御
    func showRewarded(completion: (Bool) -> Void)
}

ビジネスルール

  • Banner はプレミアム時に非表示（高さ0）
  • Interstitial は表示頻度を抑制（プレミアム時は完全非表示）
  • Rewarded は救済機能として全ユーザー利用可
  • 本番リリースは本番AdMob IDに切替必須

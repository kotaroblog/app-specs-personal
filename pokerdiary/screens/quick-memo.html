クイックメモ

席を外す前の5秒。ホールカード2枚＋任意ひとこと

設計確定・実装未着手 画面ID: PD-05

目的・役割

プレイ中に席を外す前の5秒で「今の1手」を取りこぼさないための最小入力。卓上スマホNGの制約下で唯一許容する即時記録であり、ホールカード2枚だけで保存できる軽さを最優先する。保存後は下書き（entryModeRaw="quick"）として履歴に入り、帰宅後にセッション紐づけ・詳細化（PD-04の項目追記）ができる。

UI 構成

要素種類説明
案内バナーBanner「席を外す前に5秒で。詳細はあとで足せます。」
ホールカード52セルグリッドピッカー2枚選択（PD-04と同じピッカー）。プレビューにカード表示
ひとことText Field（完全任意）1行メモ（例: BTNからの3betにAKsでオールイン、相手QQ）
保存ボタンPrimary Button「保存（あとで詳しく）」。カード2枚だけで保存可能なことをUI上で明示
補足Caption「保存後、履歴の『下書き』に入ります」

状態

状態表示内容
カード2枚選択済み保存可能。ひとことは空でよい（完全任意を明示）
保存後entryModeRaw="quick" の下書きとして履歴に格納。セッション未紐づけ（session=nil）
後日詳細化履歴の下書きから PD-04 の項目（ボード・ポジション・状況・タグ等）を追記し、セッションに紐づけ可能

遷移

操作遷移先
保存 タップホーム（PD-01）へ戻る（下書きとして履歴に格納）
キャンセル タップ呼び出し元へ戻る
（後日）履歴の下書きを開くハンド簡易記録（PD-04）で詳細化

使用するデータモデル

Hand // entryModeRaw = "quick"（5秒メモ）
     // holeCard1 / holeCard2（"Ah" 形式・必須）
     // note（1行・任意）
     // session? = nil（後からセッション紐づけ可）
     // boardCards / positionRaw / scenarioRaw 等は後日 PD-04 で追記

KPI イベント

イベントパラメータ目的
hand_loggedentry_mode=quick, tag_count, has_sessionプレイ中の即時記録量。カード内容・メモ本文は含めない

実装メモ

  • カード2枚だけで保存可能であることをボタン文言・補足でUI上明示し、5秒完結を担保する
  • ひとことは完全任意（プレースホルダーで任意と分かるようにする）
  • 保存レコードは entryModeRaw="quick" の下書き。履歴で「下書き」として区別表示し、詳細化・セッション紐づけ導線を用意
  • クイックメモとハンド簡易記録は同一 Hand モデルに統合（独立モデルにしない）。詳細化は同一レコードの編集で実現

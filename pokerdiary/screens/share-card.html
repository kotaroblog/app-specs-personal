シェアカード生成

1080×1080のハンドカードを生成して共有

設計確定・実装未着手 画面ID: PD-13

目的・役割

ハンドを1080×1080の画像カードにして共有するための画面。金額・BB収支を載せず「議論のための1手」を共有する設計とし、ストア外流入（GTM）の主導線とする。無料版はウォーターマーク付き、Proで除去できる。

UI 構成

要素種類説明
フォーマット表記Caption「1080×1080 ・ ハンドカード」。v1.0はこの1フォーマットのみ
カードプレビューShare Cardポジション＋状況・ホールカード大・ボード・メモ・右下ウォーターマーク「ポーカー手帳」
WM除去導線Text Link（Pro）「ウォーターマークを外す（Pro）」（source=shareWatermark）
共有ボタンPrimary ButtonOS共有シートを開く
設計注記Caption「金額・BB収支は載せません。誇示ではなく議論のための1手を共有する設計」

状態

状態表示内容
無料ウォーターマーク「ポーカー手帳」を右下に表示
Proウォーターマークなしで生成
共有ImageRendererで画像化しOS共有シートへ

遷移

操作遷移先
「共有する」タップOS共有シート
「ウォーターマークを外す（Pro）」タップPaywall（source=shareWatermark）

使用するデータモデル

Hand // holeCard1,2 / boardCards / positionRaw / scenarioRaw / note（カード描画用）
// 金額・BB収支（Session.spentYen / bbProfit）はカードに含めない

KPI イベント

イベントパラメータ目的
share_card_createdなしGTM素材の生成数
share_card_sharedなし実共有数（ストア外流入の種）

実装メモ

  • v1.0はハンドカード1080×1080の1フォーマットのみ（週次サマリ等はv1.1）
  • 金額・BB収支は載せない（4+品位・誇示でなく議論のための共有）
  • メモは問いかけ形式が映える（例: 「この状況、3ベットに回すべき？」）。プレースホルダー等で誘導
  • WM除去はPro（source=shareWatermark）
  • ImageRenderer（WorkoutDiaryのシェアカード実装を流用）で生成 → OS共有シート

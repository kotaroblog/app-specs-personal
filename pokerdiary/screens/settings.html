設定

記録・プラン・アプリの3グループ

設計確定・実装未着手 画面ID: PD-16

目的・役割

店マスタ・通知・プラン・法務情報を集約する設定画面。記録／プラン／アプリの3グループに整理し、Proアップグレード（source=settings）や店マスタ管理・週次レビュー通知の設定を提供する。

UI 構成

グループ項目説明
記録店マスタを管理Venueの追加・編集・削除
記録週次レビュー通知曜日／時刻の設定（既定: 日 21:00）
プラン現在のプラン無料／Proの表示
プランProにアップグレードPaywall（source=settings）へ
プラン購入を復元過去購入の復元
アプリ利用規約app-policies の規約ページ
アプリプライバシーポリシーapp-policies のプライバシーページ
アプリ特定商取引法に基づく表記app-policies の特商法ページ
アプリバージョンアプリバージョン表示

状態

状態表示内容
無料ユーザー現在のプラン=無料。「Proにアップグレード」を表示
Proユーザー現在のプラン=Pro
DEBUGビルドアプリグループ末尾に EventLogDebugView を追加表示

遷移

操作遷移先
店マスタを管理 タップ店マスタ管理
週次レビュー通知 タップ通知設定（曜日・時刻）
Proにアップグレード タップPaywall（source=settings）
各法務リンク タップapp-policies の各ページ

使用するデータモデル

Venue // 店マスタ管理の対象
// 通知設定・プラン状態は UserDefaults / SubscriptionManager で管理（SwiftDataモデルは持たない）

実装メモ

  • 3グループ（記録／プラン／アプリ）で構成
  • Proアップグレードは source=settings でPaywall（PD-14）へ
  • 法務リンクは特定商取引法を含む（審査要件）
  • DEBUGビルドのみ、アプリグループ末尾に EventLogDebugView（旧PokerLesson流用）を表示
  • 週次レビュー通知の既定は日曜21:00。曜日・時刻を変更可能

ASOキーワード戦略

App Store Optimization の方針（要レビュー）

ターゲットユーザー

  • ペルソナ中学生〜大学生、社会人の資格学習者
  • 課題学習時間を可視化したい、教科別に管理したい、モチベを保ちたい
  • 利用シーン受験勉強、定期テスト対策、資格学習、自宅・図書館・学校

説明文の構造

  1. 「教科ごとに勉強時間を測って、可視化」
  2. Live Activity でロック画面表示
  3. 機能リスト: タイマー / ヒートマップカレンダー / 週間レポート / Widget / Live Activity / SNS共有
  4. 使い方: 教科を選んで開始ボタン
  5. 差別化: Live Activity / Instagram Stories共有 / ポモドーロ自動切替

StudyStopwatch

学習タイマー / Live Activity / Widget · Bundle ID: com.happyboy1002.StudyStopwatch

概要

  • カテゴリ教育
  • 対応OSiOS 18.0+ / Swift 6.0
  • 言語日本語のみ
  • マネタイズサブスクリプション + AdMob広告
  • ステータスリリース済
  • バージョンv1.0.5 (build 6)
  • 表示名学習ストップウォッチ
  • 拡張ターゲットLive Activity / Widget
  • App Groupgroup.com.happyboy1002.StudyStopwatch

アプリの目的

教科ごとに学習時間を計測し、Live Activityでロック画面・Dynamic Islandに表示。 ヒートマップカレンダー・週間/月間レポート・Instagram Stories共有でモチベ維持を支援。 ポモドーロモード（集中25分/休憩5分）対応。

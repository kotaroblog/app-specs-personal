目標・やることリスト画面（GoalEditSheet）

今日 / 今週 / 今月の目標時間設定 + やることリスト管理

v1.1 画面ID: SS-GOAL-01

目的・役割

ユーザーが「今日 / 今週 / 今月」単位の学習目標時間を設定し、あわせてやることリスト（StudyTask）を管理するシート。 ホーム画面の CompactGoalCard からモーダルとして呼び出される。 v1.1 でやることリストセクションを追加し、計画→実行→記録の流れをアプリ内で完結できるようにした。

UI 構成

要素種類説明
期間タブセグメントコントロール今日 / 今週 / 今月 の切り替え
目標時間スライダーSlider + TextField分単位で入力。今日は 1〜480 分、今週・今月は上限を自動スケール
やることリストList セクション選択期間に紐づく StudyTask の一覧（後述）
保存ボタンButton（.prominent）目標時間を UserSettings に保存してシートを閉じる

やることリストセクション

要素説明
タスク行（チェックボックス）タップでシンプル完了。isCompleted = true、completedAt = now をセット
タスク行（「📝 記録して完了」ボタン）科目選択 → ManualStudyEntrySheet 起動 → 保存と同時にタスク完了。StudySession.id を linkedSessionID に格納
タスク行（タイトルタップ）インライン編集 TextField を展開してタイトルを変更
タスク行（trash アイコン）タスク削除（確認アラートなし、即時削除）
追加 TextField「やることを入力」プレースホルダー。入力なしの場合、追加ボタンは disabled グレー
追加ボタンTextField が空でなければ新規 StudyTask を生成・保存
完了率タイル（週/月のみ）今週 / 今月の「完了率% ・ 完了数 / 全体数」を集計カードとして上部に表示

状態

状態表示内容
タスクなし「やることをリストに追加しましょう」空状態メッセージ
全タスク完了「すべて完了！」メッセージ + チェックマークアイコン
「記録して完了」→ ManualStudyEntrySheet 起動中シートが前面に表示される。キャンセル時はタスク未完了のまま

遷移

操作遷移先
「📝 記録して完了」タップManualStudyEntrySheet（モーダル）。onSaved コールバックでタスク完了フラグをセット
保存 / キャンセル呼び出し元（ホーム画面 / カレンダー画面）に戻る

使用するデータモデル

// 目標時間（UserSettings）
@Model final class UserSettings {
    var dailyGoalMinutes: Int?
    // 週次・月次目標は今後追加予定
}

// やることリスト
@Model final class StudyTask {
    var id: UUID = UUID()
    var periodRawValue: String = "day"   // "day" / "week" / "month"
    var title: String = ""
    var isCompleted: Bool = false
    var periodStart: Date = Date()
    var periodEnd: Date = Date()
    var linkedSessionID: UUID?           // 「記録して完了」時に紐づく StudySession.id
    var completedAt: Date?
    var sortOrder: Int = 0
    var createdAt: Date = Date()
}

実装メモ

  • GoalEditSheet は .sheet モディファイアで表示。期間タブは @State var selectedPeriod: TaskPeriod
  • StudyTask の @Query は periodRawValue == selectedPeriod.rawValue AND periodStart <= now AND periodEnd >= now でフィルタ
  • 「記録して完了」は ManualStudyEntrySheet に initialNote + onSaved クロージャを渡し、保存完了後にタスクの isCompleted を true にセット
  • タスク追加時の periodStart / periodEnd は selectedPeriod に応じて今日の0時〜23:59:59 / 今週月曜〜日曜 / 今月1日〜末日 で算出
  • trash アイコン削除は modelContext.delete(task) を即時実行

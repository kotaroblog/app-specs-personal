科目編集（SubjectEditView）

科目の追加・編集

v1.0画面ID: SS-SUBJ-01

UI 構成（v1.1）

v1.2.2 で点数目標セクションを追加。オンボーディング必須フローからの移設により、既存ユーザーを含む全ユーザーがこの画面から点数目標を設定・更新できるようになった。
要素説明
名前入力TextField（必須）
カラー選択カラーピッカー
アイコン選択SF Symbol一覧
並び順Stepper or Drag&Drop
点数目標セクション（追加）前回点（currentScore）・目標点（targetScore）の2フィールド（いずれも任意・0〜100）。goalType 問わず全ユーザーに表示
削除ボタン確認後 cascade で関連 StudySession も削除

点数目標の表示・保存ルール

  • currentScore / targetScore ともに任意（空のまま保存可能）
  • 入力範囲: 0〜100（整数）。範囲外入力時はバリデーションエラー表示
  • 保存したスコアはレポートタブの成績グラフ（SS-GRADES-01）で参照される
  • v1.2.2 以前にオンボーディングで入力済みのスコアはそのまま引き継がれる

使用するデータモデル

Subject { id, name, colorHex, iconName, sortOrder, currentScore?: Int?, targetScore?: Int? }

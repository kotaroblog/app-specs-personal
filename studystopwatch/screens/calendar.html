カレンダー画面（CalendarScreen）

日 / 週 / 月の3ビュー切替で学習実績を可視化するリニューアル版カレンダー

v1.0 画面ID: SS-CAL-02

目的・役割

旧 CalendarHeatmapView（SS-CAL-01）を置き換える新カレンダー画面。 日・週・月の3ビューをタブで切り替え、試験日カウントダウン・学習タイムライン・月次ヒートマップを統合し、 ユーザーが自身の学習リズムを多角的に把握できるようにする。 やることリスト（StudyTask）との連携により、計画と実績を同一画面で確認できる。

画面構成概要

コンポーネント種類説明
CountdownCardカード試験日までのカウントダウン（UserSettings.examDate 連携）。試験日未設定時は非表示
ViewModeTabsセグメントコントロール今日 / 週 / 月 の3タブ。タップで即切替
DayCalendarViewスクロールビュー「今日」タブの中身（→ 後述）
WeekCalendarViewスクロールビュー「週」タブの中身（→ 後述）
MonthCalendarViewグリッド + 展開領域「月」タブの中身（→ 後述）

DayCalendarView（今日ビュー）

要素説明
ミニ週ストリップ当週7日を横並び。タップした日付を selectedDate に更新。当日は accent 色の円形バッジ
日付ヘッダ + ナビ「YYYY年MM月DD日（曜）」表示。左右シェブロンで前後1日移動
ヒーローカード緑グラデ背景。選択日の合計学習時間・日次目標・達成率（%）を大きく表示
やることリストその日付に紐づく StudyTask（period=day）のリスト。チェック・編集・削除・追加TextField 付き
科目別実績バー科目ごとの当日学習時間を横バーで表示（Subject.colorHex）

WeekCalendarView（週ビュー）

要素説明
週サマリカード週合計時間 / 週次目標 / 達成率リングを上部に表示
週次タイムライン月曜始まり7列 × 06時〜23時（hour height = 40pt）の縦グリッド。StudySession を科目色のブロックで描画
今日列ハイライト今日の列に accent 5% の背景色を適用
現在時刻ライン（NOW）今週のみ表示。赤い横線で現在時刻を示す
週次科目別積み上げ今週の科目別累積時間を横長バー + 2列グリッドで一覧表示

MonthCalendarView（月ビュー）

要素説明
7×6 グリッド月曜始まり。当月外の日はグレーアウト
今日セルaccent 色の円形バッジで日付を強調
曜日色分け日曜 = #E85D75、土曜 = #4A90E2
科目色バー各セル上部に学習した科目の色バー（最大3科目）
実績ドット各セル下部に StudySession の存在を示すドット（最大3）
次回予定ピン次回予定日セルの右上にピンアイコンを表示
日タップ展開タップで selectedDate を更新 → グリッド下に DayCalendarView をインライン展開

状態

状態表示内容
選択日に記録なし（日ビュー）ヒーローカードに「0時間 0分」、科目別実績なしメッセージ
今週に記録なし（週ビュー）タイムライン空白。週サマリは0表示
試験日未設定CountdownCard を非表示
やることリスト空（日ビュー）「やることを追加しましょう」プレースホルダー

遷移

操作遷移先
タイムラインブロックタップ（週ビュー）セッション詳細シート（モーダル）
「📝 記録して完了」タップ（やることリスト）ManualStudyEntrySheet（モーダル）
月ビューで日タップ同画面内 DayCalendarView をインライン展開
CountdownCard の試験日タップ設定画面（試験日設定セクション）

使用するデータモデル

// 学習セッション
@Model final class StudySession {
    var id: UUID
    var startTime: Date
    var endTime: Date?
    var duration: TimeInterval
    var subject: Subject?
}

// 科目
@Model final class Subject {
    var id: UUID
    var name: String
    var colorHex: String
}

// ユーザー設定（試験日カウントダウン用）
@Model final class UserSettings {
    var dailyGoalMinutes: Int?
    var examDate: Date?
}

// やることリスト（日ビューで利用）
@Model final class StudyTask {
    var id: UUID
    var periodRawValue: String   // "day" / "week" / "month"
    var title: String
    var isCompleted: Bool
    var periodStart: Date
    var periodEnd: Date
    var linkedSessionID: UUID?
    var completedAt: Date?
    var sortOrder: Int
    var createdAt: Date
}

実装メモ

  • 旧 CalendarHeatmapView / DayDetailSheet は削除済み。SS-CAL-01 の画面IDは旧仕様として残す
  • CalendarScreen が親 View。ViewModeTabs の選択状態を @State で保持
  • selectedDate は CalendarScreen が管理し、各子ビューに Binding で渡す
  • WeekCalendarView のタイムラインは ScrollView で縦スクロール。水平は固定7列（GeometryReader で列幅算出）
  • MonthCalendarView の inline 展開は AnyLayout / if selectedDate != nil で制御
  • テーマカラーは AppTheme.accent（#58CC02）。カードは cardShadow() 修飾子で統一
  • StudyTask の @Query は periodStart/periodEnd で当日範囲をフィルタ

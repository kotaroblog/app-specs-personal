カレンダー画面（CalendarScreen）

日 / 週 / 月の3ビュー切替で学習実績と期限タスクを一体管理するカレンダー

v1.3.0 画面ID: SS-CAL-02

目的・役割

旧 CalendarHeatmapView（SS-CAL-01）を置き換える新カレンダー画面（v1.1.0 で初導入、v1.2.0 で大幅刷新）。 日・週・月の3ビューをタブで切り替え、試験日カウントダウン・学習タイムライン・月次グリッドを統合する。 v1.2.0 で期限日ベースの StudyTask と完全統合し、計画（タスク）と実績（学習時間）を同一画面で管理できるようにした。 また学習計画自動配分（StudyPlanSheet）へのアクセス導線を右上ツールバーに追加した。

使用するデータモデル

// 学習セッション
@Model final class StudySession {
    var id: UUID
    var startTime: Date
    var endTime: Date?
    var duration: TimeInterval
    var subject: Subject?
}

// 科目
@Model final class Subject {
    var id: UUID
    var name: String
    var colorHex: String
}

// ユーザー設定（試験日カウントダウン用）
@Model final class UserSettings {
    var dailyGoalMinutes: Int?
    var examDate: Date?
}

// やることリスト（v1.2〜: 期限日ベース）
@Model final class StudyTask {
    var id: UUID
    var title: String
    var isCompleted: Bool
    var dueDate: Date?            // v1.2〜: 期限日（日ビューは dueDate == selectedDate でフィルタ）
    var linkedSessionID: UUID?
    var completedAt: Date?
    var sortOrder: Int
    var createdAt: Date

    // 後方互換フィールド（v1.2 以降は非推奨）
    var periodRawValue: String    // 旧: "day" / "week" / "month"
    var periodStart: Date
    var periodEnd: Date
}

実装メモ

  • 旧 CalendarHeatmapView / DayDetailSheet は削除済み。SS-CAL-01 の画面IDは旧仕様として残す
  • CalendarScreen が親 View。ViewModeTabs の選択状態を @State で保持
  • selectedDate は CalendarScreen が管理し、各子ビューに Binding で渡す
  • WeekCalendarView のタイムラインは ScrollView で縦スクロール。水平は固定7列（GeometryReader で列幅算出）
  • MonthCalendarView の inline 展開は AnyLayout / if selectedDate != nil で制御
  • テーマカラーは AppTheme.accent（#58CC02）。カードは cardShadow() 修飾子で統一
  • 日ビューの StudyTask @Query: dueDate が selectedDate と同日のタスクを取得（startOfDay/endOfDay で範囲フィルタ）
  • 週ビュー①「今日のやること」: dueDate == 今日 の未完了タスクを取得してチェックボックスリストで表示
  • 週ビュー①「取りこぼし」: dueDate < startOfDay(today) かつ isCompleted == false のタスクをオレンジグループで表示
  • 週ビュー②「今週の進捗」: dueDate が今日〜今週日曜のタスクを subject でグルーピングし SubjectProgressRow で表示。日後ラベルは全廃
  • 月ビューのタスクバッジ: 月内全タスクを取得し、日付ごとに Dictionary でグルーピングして未完了件数を算出
  • StudyPlanSheet は .sheet(isPresented: $showPlanSheet) で表示
  • CountdownCard の残り日数: Calendar.current.dateComponents([.day], from: today, to: examDate).day で算出

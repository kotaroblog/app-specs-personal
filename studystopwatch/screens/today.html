今日画面（TodayView）

本日の学習ダッシュボード

v1.0画面ID: SS-TODAY-01

目的・役割

本日の学習時間・達成率・科目別内訳を一目で確認できるダッシュボード画面。

UI 構成（UI/UX改善版）

v1.2.1 でやることセクションを追加し、タスク追加・完了の導線を Today 画面単独で完結できるよう強化。
要素説明
達成率リングdailyGoalMinutes に対する当日達成率をリング形式で表示
本日の合計時間大きく中央表示
科目別サマリ教科ごとの時間内訳（Subject.colorHex の色バー）
前日比「前日より N% UP/DOWN」
平均セッション時間本日のセッション平均
やることセクション（追加）本日期限（dueDate == 今日）の StudyTask をチェックボックスリストで表示。タスク追加フィールドとタスク作成シートへの導線付き
タスク追加フィールドインライン TextField + 送信で即時 StudyTask 生成（期限日 = 今日）
「期限付きで追加」ボタンTaskCreationSheet を表示（日付ピッカーで期限日を変更可）
シェアボタンStudyShareSheet 起動（ツールバー trailing または画面下部）

状態

状態表示
記録あり達成率 + 科目別内訳 + やることリスト
記録なし達成率リング 0% + 「記録を始めよう」CTA（ホームタブへ誘導）
目標達成達成バッジ表示
今日のタスクなしやることセクションに「今日やることを追加しましょう」プレースホルダー + 追加フィールド

遷移

操作先
シェアStudyShareSheet（sheet）
「期限付きで追加」ボタンTaskCreationSheet（create モード、期限日 = 今日）
タスクタイトルタップ（編集）TaskCreationSheet（edit モード）

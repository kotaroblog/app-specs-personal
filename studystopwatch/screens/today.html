今日画面（TodayView）

本日の学習ダッシュボード

v1.0画面ID: SS-TODAY-01

目的・役割

本日の学習時間・達成率・科目別内訳を一目で確認できるダッシュボード画面。

UI 構成（v1.2.2）

v1.2.2 で表示優先順位を再設計。「今日やること → 今日の実績（合計時間＋目標残り）→ 科目別内訳」の順に並べ替え。 グラフ・タイムライン・セッション履歴は詳細折りたたみセクションへ移動。前日比に増減時間を追加。 実績カードは重要カードスタイル（太い枠線＋底面シャドウ）を適用。
順序要素説明
1今日やることセクション（todayTasksCard）重要カードスタイル。本日期限（dueDate == 今日）の StudyTask をチェックボックスリストで表示。タスク追加フィールドとタスク作成シートへの導線付き
2今日の実績カード（todayAchievementCard）重要カードスタイル。合計時間を大きく中央表示 + 目標残り時間。達成率リング（effectiveDailyGoalMinutes 基準）
3前日比「前日より N% UP/DOWN」に加えて増減時間も併記（例: +31% / 前日比 +25分）
4科目ごとの内訳教科ごとの時間内訳（Subject.colorHex の色バー）
5「詳細を見る」折りたたみセクショングラフ・タイムライン・セッション履歴・平均セッション時間を格納。デフォルト折りたたみ
—タスク追加フィールドインライン TextField + 送信で即時 StudyTask 生成（期限日 = 今日）
—「期限付きで追加」ボタンTaskCreationSheet を表示（日付ピッカーで期限日を変更可）
—シェアボタンStudyShareSheet 起動（ツールバー trailing）

状態

状態表示
記録あり今日やること → 今日の実績 → 前日比 → 科目別内訳 → 詳細折りたたみ
記録なし今日の実績カード（合計0分）+ 「記録を始めよう」CTA（ホームタブへ誘導）
目標達成達成バッジ表示（実績カード内）
今日のタスクなし「今日やることを追加しましょう」プレースホルダー + 追加フィールド
詳細展開「詳細を見る」セクションをタップでグラフ・タイムライン・履歴を展開

遷移

操作先
シェアStudyShareSheet（sheet）
「期限付きで追加」ボタンTaskCreationSheet（create モード、期限日 = 今日）
タスクタイトルタップ（編集）TaskCreationSheet（edit モード）

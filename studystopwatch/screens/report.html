レポート画面（ReportView）

週間・月間・通期の統計レポート + 成績セグメント

v1.2.0 画面ID: SS-REPORT-01

目的・役割

期間別の学習統計を表示する。上部 SegmentedPicker（学習時間 / 成績）で2セクションを切り替える。 学習時間セグメントには v1.2.0 で「先週のふりかえり」カード（WeeklyCoachReportCard）を追加。 成績セグメントは grades-tab.html を参照。

UI 構成（学習時間セグメント）

要素説明
セグメントPicker「学習時間」/ 「成績」（上部に常時表示）
期間Picker週 / 月 / 通期
合計学習時間選択期間のトータル
先週のふりかえりカードWeeklyCoachReportCard。Premium = サマリー+コーチメッセージ、無料 = blur+ロック誘導（v1.2.0 新規）
詳細統計（Premium）前週比 / 1日平均 / ベスト曜日。FeatureGate.canAccessPremiumStats が false なら非表示+ロックUX
日別折れ線グラフ日別の推移（SwiftUI Charts）
科目別バー科目ごとの時間内訳
シェアボタンtoolbar trailing。学習時間セグメント時のみ表示（成績セグメントでは非表示）

先週のふりかえりカード（WeeklyCoachReportCard）

要素無料プランPremiumプラン
カード全体blur(radius: 6) でぼかし通常表示
表示内容ロック画面（lock.fill アイコン + 「先週のふりかえりを解放」）週合計時間 + 前週比 + 勉強日数 + 最多科目 + 試験日残日数 + コーチメッセージ
コーチメッセージ—達成度に応じた3段階（75%以上 / 50〜75% / 50%未満）。目標未設定時は勉強日数で判定
タップweeklyReportLockTapped 計測 → PremiumView(source: .weeklyReport)weeklyReportViewed 計測

状態

条件表示
無料基本統計 + ふりかえりカード（blur+ロック） + 詳細統計ロック
Premium基本統計 + ふりかえりカード（全表示） + 詳細統計表示
前週レポートが未生成（記録ゼロ週）ふりかえりカード非表示（WeeklyCoachReport が nil の場合は表示しない）

遷移

操作遷移先
シェアボタンをタップStudyShareSheet（sheet）
詳細統計ロックをタップPremiumView(source: .reportStats)（sheet）
ふりかえりカードロックをタップPremiumView(source: .weeklyReport)（sheet）
セグメントを「成績」に切り替えGradeReportView を表示（同一 NavigationView 内）

使用するデータモデル

  • StudySession（@Query で期間フィルタ）
  • Subject（科目別集計）
  • WeeklyCoachReport（@Query sort: weekStart desc、最新1件を表示）
  • UserSettings（目標分数・試験日）

実装メモ

  • @Query(sort: \WeeklyCoachReport.weekStart, order: .reverse) private var weeklyReports: [WeeklyCoachReport] で取得し、weeklyReports.first を WeeklyCoachReportCard に渡す
  • WeeklyCoachReportCard は FeatureGate.shared.canAccessWeeklyCoachReport で分岐
  • ReportSection enum: .time / .grade。学習時間セグメント時のみ toolbar に共有ボタンを表示
  • 期間別フィルタ（週/月/通期）は StudySession の startTime でフィルタし SwiftUI Charts で描画

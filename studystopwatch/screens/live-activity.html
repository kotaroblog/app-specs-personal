Live Activity（StudyStopwatchLiveActivity）

ロック画面・Dynamic Islandタイマー表示

v1.1画面ID: SS-LA-01

表示モード

モード表示内容
Lock Screen科目アイコン + 科目名 + 経過時間（大）
Dynamic Island Compact (Leading)科目アイコン
Dynamic Island Compact (Trailing)経過時間
Dynamic Island Expanded科目名 + 経過時間 + ストリーク
Dynamic Island Minimal科目アイコン

データソース（v1.1）

struct StudyActivityAttributes: ActivityAttributes {
    var subjectName: String
    var subjectColorHex: String
    var subjectIconName: String
    var startTime: Date

    struct ContentState: Codable {
        var anchorDate: Date            // 計測開始の基準日時（=現在時刻−累積経過秒）
        var pausedElapsedSeconds: Int?  // 一時停止中の凍結表示用経過秒。nil=計測中
        var subjectName: String
    }
}
経過時間は anchorDate を起点にシステムが Text(timerInterval:) で自動表示する。 一時停止中は pausedElapsedSeconds の値を静的表示する（nil のときは計測中＝自動更新）。

更新タイミング（v1.1）

  • タイマー開始時 → Activity 起動（anchorDate をセット）
  • 状態変化時（一時停止 / 再開 / フェーズ切替）→ ContentState を更新
  • 計測中の経過時間 → システムが anchorDate から自動更新（アプリ側の毎秒更新は行わない）
  • タイマー停止時 → Activity 終了

通知機能

日次リマインダー（ローカル通知）

v1.0機能ID: SS-FEAT-NOTIF

概要

毎日定時に「今日の勉強は始めましたか？少しずつ頑張りましょう！」のリマインダーを送る機能。 v1.0.5から復習リマインダー（学習セッション終了後に指定時間後へ通知）を追加。 ローカル通知のみで、サーバー不要。

通知種別

種別通知ID説明追加バージョン
デイリーリマインダー"dailyReminder"毎日定時に学習開始を促すv1.0
復習リマインダー"reviewReminder_[sessionID]"セッション終了後N時間後に「復習しよう」通知v1.0.5

NotificationService API

final class NotificationService {
    static let shared: NotificationService

    func requestAuthorization() async throws
    func setEnabled(_ enabled: Bool, hour: Int, minute: Int)
    func scheduleDailyReminder(hour: Int, minute: Int)

    // v1.0.5追加
    func scheduleReviewReminder(sessionID: String, subjectName: String, afterHours: Int)
    func cancelReviewReminder(sessionID: String)
}

処理フロー（デイリーリマインダー）

  1. 初回 ON 時に requestAuthorization([.alert, .sound, .badge])
  2. 許可されたら UNCalendarNotificationTrigger を作成
  3. 毎日その時刻に通知配信
  4. OFF 時に removePendingNotificationRequests(withIdentifiers:)

処理フロー（復習リマインダー）（v1.0.5追加）

  1. セッション完了時に scheduleReviewReminder(sessionID:subjectName:afterHours:) を呼び出し
  2. UNTimeIntervalNotificationTrigger で指定秒数後に配信
  3. 通知文面: 「[科目名] の復習をしましょう！記憶の定着には繰り返しが大事です。」
  4. ユーザーが設定で OFF にすると既存の pending 通知もキャンセル

ビジネスルール（v1.0.5）

  • 復習リマインダーはプレミアム限定（無料ユーザーは動画ボーナス適用中のみ利用可）
  • 間隔設定: 1時間 / 3時間 / 6時間 / 12時間 / 24時間 から選択
  • 同一セッションへの復習リマインダーは1件のみ（重複スケジュール不可）

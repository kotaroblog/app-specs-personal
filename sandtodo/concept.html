1. コンセプト（再定義）

ポジショニング

SandDone は「タスク管理アプリ」ではない。「感情演出アプリ・気持ちの整理アプリ」として位置づける。Todoist / TickTick とは競合しない。競合するのは Forest / Calm / Headspace である。

カテゴリ代表プロダクト関係
管理系Todoist / TickTick / Things 3競合しない（別ジャンル）
感情系Forest / Calm / Headspace直接競合
Aesthetic系Things 3 / Streaks隣接（重なるユーザー層）

一文要約

完了した言葉が文字のまま画面に積もり、1日の終わりに静かに埋葬される。気持ちの整理を「視覚と触感」で体験するアプリ。

キャッチコピー

「こなした言葉が、砂になって積もる」

中核アイデア

  • 完了 = 文字になって積もる（「消える」でも「崩れる」でもない）
  • 1日の終わりに「埋葬」演出、翌朝は「風化」した残骸から再開
  • 状態変化を感情で理解させるUI。物理演算はリアルにしすぎない
  • 文字山は AI 画像ではなく 文字山方式（PoC 動作確認済み）で実装

2. アプリ名: SandDone

変更理由

  • Done = 完了感がコンセプト（完了した言葉が積もる）と直結する
  • 海外発音しやすく ASO で有利
  • GPT・Claude の両 AI 評価で推奨

3. 体験フロー（短縮・操作分離版）

タイムライン（1タスク完了時）

0.00s  タップ + Haptic + 砂山が一瞬縮む（インパクト感）
0.18s  カード沈み
0.33s  カード崩壊 + 文字バラバラ
0.55s  ★ 次タスク操作可能（世界と操作の分離）
0.55s〜1.5s  文字が画面下へ落下
1.5s   文字山に着地 + データ削除 + 残タスク再配置

重要な設計変更: 操作の分離

演出と操作を分離することで「5件連続完了」の拘束時間を大幅短縮:

項目旧（v1.x）新（v2.0）
演出全体時間2.3 秒1.5 秒
次タスク操作可能2.3 秒後0.55 秒後
5件連続完了の拘束約 11.5 秒約 3 秒

「世界（文字落下）は続いているが、ユーザーは次の操作ができる」という分離がストレスゼロの体験を生む。

6. 文字山の物理（実装方針）

基本方針（PoC 動作確認済み）

  • 各文字を独立した Particle として管理
  • 重力加速度: 260
  • 安息角チェックで自然な山形を形成
  • ヒープ着地（先に落ちた文字の上に積み上がる）

パフォーマンス最適化（v2.0 追加）

  • iPhone SE / 低電力モード時: 粒子数を自動削減（最大 50 → 30 に絞る）
  • Dynamic Type 対応: フォントサイズ変化に合わせて粒子サイズを追従
  • 日本語長文の自動縮小: 文字列が 10 文字を超える場合は縮小スケール適用

7. 長期利用設計（v2.0 重要追加）

日々の変化

時間帯状態・演出
朝風化した残骸・薄い風紋が残る（昨日の跡）
昼タスク完了のたびに文字が積もる
夕方〜夜山が完成形に近づく
夜（任意）「今日を埋葬」演出 — ユーザーが意識的に1日を締める
翌朝残骸（未完了の痕跡）を残して再開

「山を崩す」儀式機能

  • 週次・月次で「砂山を崩す」アクション
  • 積み上がった言葉が一斉に崩れ流れる演出
  • 「リセット」ではなく「儀式」として設計（達成感を強調）

8. ペルソナ

共通特性: 「タスク管理ユーザー」ではなく「気持ちの整理を求めるユーザー」

A: 焦りを感じるサラリーマン

  • 年齢・属性 28〜35歳 IT / 商社 / メーカー
  • Todoist 管理疲れ、無機質なタスクツールに感情が乗らない
  • 求めていること: 重荷が消える体験、感情的な解放感、1日の終わりの区切り

B: 自己管理に悩む若手社会人・大学生

  • 年齢・属性 20〜25歳
  • Forest / Habitica を 3 日で辞める。Things 3 は ¥1,200 で高い
  • 求めていること: 可愛さ・気持ちよさ重視、SNS でシェアしたい体験

C: 副業フリーランス 30 代前半

  • 年齢・属性 30〜35歳 エンジニア / ライター / デザイナー
  • Notion は重い、タスク管理より「気持ちの切り替え」がしたい
  • 求めていること: 1 日終わりの切り替えルーティン、シンプルさ、美しさ

価格根拠

Forest ¥650 / Things 3 ¥1,200 / CHROMATIC ¥240。¥480 はインパルス購入の上限ラインであり「安すぎ = 信頼欠落」のリスクを回避する価格帯。

AdMob バナー非採用の理由

バナーはミニマル UI・砂の世界観と致命的に相性が悪い。ユーザー体験の破壊は長期継続率の低下に直結するため不採用とする。

LTV 試算

  • 課金転換率: 5%
  • 買い切り単価（Apple 手数料 15% 控除後）: ¥408 / 人
  • 100 人課金 → 月収 ¥40,800 相当

12. 市場性

  • 年間売上目標への寄与中〜高
  • 期待月収益1.5〜3.0 万円
  • SNS 拡散ポテンシャル高（文字山動画は TikTok で強い）

注意: ToDo 業界の 30 日継続率は 10% 以下が相場。長期設計（風化・埋葬・月次儀式）が差別化の鍵。

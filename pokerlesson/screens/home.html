ホーム画面

アプリ起動後の最初の画面。プロフィール・統計・コース絞り込み・レッスン開始・今日の1問・他機能メニューを縦リストに集約する。

画面ID: HOME-01 v1.1 リデザイン反映

目的・役割

ユーザーが「今日何をするか」を素早く選択できる起点画面。 v1.1 で実務UIにリデザインし、ヒーロー演出（ブランドコピー・TABLE 番号・ROOM 名）を排して、 情報密度と継続使用に耐える静かなダッシュボードに統一した。

UI 構成（v1.1 縦リスト）

すべて ScrollView 内の VStack(spacing: 16) に格納。navigationTitle("ポーカーレッスン") をインライン表示。

順序要素種類説明
1 プロフィールヘッダー Card（goldCard） アバター（ランクアイコン）・ニックネーム編集 button・ランクバッジ・XPバー
2 統計カード Card（3カラム） 回答数 / 正答率 / 連続日数。1pt縦divider 区切り、数値は monospaced
3 フィルタ Card（chip×2列） コース（すべて / RFI / vs Open / Squeeze）・難易度（すべて / 基礎 / 標準 / 応用）の選択chip
4 レッスン開始 CTA Primary Button 「レッスンを開始」 + サブ「5問 / 全 N 問から出題」 + 右に arrow.forward。アクセント色塗り、白文字
5 今日の1問 カード Card（goldCard） 未完了時はアクティブ、完了時は checkmark + 「完了しました」 + 「また明日、1問。」
6 機能メニューリスト Card（リスト型） CPU対戦 / レンジ表 / 進捗 / 設定。行ごとに 1pt divider、左にSF Symbols、右に chevron
7 コーチコメント Card（条件表示） tendency.hasEnoughData 時のみ。lightbulb アイコン + 「コーチからのコメント」 + dealerNote 本文
8 無料枠カード Card（無料ユーザーのみ） 残数 monospace表示 + 上限到達時のみ「プレミアムで全問解放」CTA

v1.1 リデザインで削除した要素

  • ブランドコピー brandTagline（"LET'S PLAY ONE HAND." セリフ + 「一緒に、もっと強くなろう。」）
  • 「今日の席」演出 tableSeatHeader（"TABLE 47 KYOTO BACK ROOM" + 0.5/1 bb 表記）
  • "START LESSON" セリフ大文字 CTA + tracking 装飾（→ 通常の「レッスンを開始」CTA）
  • dealerWhisper モノスペース「ディーラーから一言」装飾（→ 通常見出しの「コーチからのコメント」）
  • "TODAY'S NOTE" モノスペース英字ラベル（→ 「今日の1問」日本語見出し）
  • ContentView での EnteringTableView 着席演出オーバーレイ呼び出し（ファイルごと削除）

状態

状態表示
問題データ読込中 CTA下に「問題データを読み込み中…」 textMuted で表示
フィルタ結果 0 件 CTA下に「選択中のコース・難易度に該当する問題がありません」表示。CTAは disabled 50% opacity
無料枠到達 無料枠カード内に「プレミアムで全問解放」CTAを表示。CTAタップで showPaywall = true
今日の1問 未完了 カード左に note.text アイコン（accentSubtle背景）、本文と chevron でタップ可能
今日の1問 完了済み checkmark（success色）+ 「完了しました」「また明日、1問。」。タップ不可（Buttonでなく HStack）
コーチコメント データ不足 カード自体を表示しない（tendency.hasEnoughData == false でスキップ）

遷移

操作遷移先条件
「レッスンを開始」CTAをタップ QuizView（filteredQuestions から5問ランダム抽出） filteredQuestions が空でない・無料枠未到達
「レッスンを開始」CTAを無料枠到達時にタップ PaywallView（source: .quizLimit） 累計回答 ≥ 30問
今日の1問カードをタップ QuizView（isDailyReview=true, dailyReviewSource付き） 未完了時のみ
機能メニュー: CPU対戦 CPUMatchView（onSelectMenu: .cpuMatch） 常時
機能メニュー: レンジ表 HandRangeView 常時（ロックポジションは Paywall）
機能メニュー: 進捗 ProgressDashboardView 常時（累計タブは Premium）
機能メニュー: 設定 SettingsView 常時
プロフィール内ニックネーム編集ボタン NicknameEditorSheet（detents: .height(220)） 常時

使用するデータモデル・サービス

// SwiftData @Query
@Query private var allQuestions: [Question]
@Query private var progressList: [UserProgress]
@Query private var allRecords: [AnswerRecord]

// AppStorage
@AppStorage("quiz_course") selectedCourseRaw     // "all" | "rfi" | "vs_open" | "cold_call_vs_squeeze"
@AppStorage("quiz_difficulty") selectedDifficulty // 0..3
@AppStorage("player_nickname") nickname
@AppStorage("daily_review_last_completed_at") dailyReviewLastCompletedAt

// 派生値
profileLevel = ProfileLevel.from(records: allRecords)
tendency     = PlayerTendency.compute(from: allRecords)

// DailyReview
DailyReview.isCompletedToday(lastCompletedAt: ...)
DailyReview.select(from: allQuestions, records: allRecords)

// イベントロギング
EventLogger.shared.log(.dailyReviewCardShown, parameters: ["source": ...])
EventLogger.shared.log(.dailyReviewStarted,    parameters: ["source": ...])
EventLogger.shared.log(.lessonStartedAfterDaily)

実装メモ

  • 削除した today_table_number / today_table_date @AppStorage は今後復活させない（演出を完全排除する方針）
  • HomeView 本体は ScrollView 内の単一 VStack。portraitLayout / landscapeLayout の分岐は v1.0 で存在したが、リデザインで縦リスト一本化
  • 各セクションは goldCard() ViewModifier（中身は 1pt ボーダー + surface 塗り）に統一
  • chip 共通スタイル：未選択 surfaceElevated + border、選択中 accentSubtle + accent 1pt + accent 文字

デザインシステム

UIトーン・カラー・タイポ・コンポーネントの正典。Linear / GitHub / Stripe Dashboard 系の実務UIに揃える。

DESIGN-01 v1.1

方向性

本アプリは「毎日使う学習ツール」として運用するため、装飾的なラウンジ表現（葉巻ラウンジ風・真鍮金・フェルトグリーン・Cinzelセリフ）は v1.1 で全廃した。 実務系SaaS（Notion / GitHub / Linear / Stripe Dashboard）と同等の落ち着いたダークUIに統一し、可読性・情報整理・操作効率を最優先とする。

カラーパレット

すべて PokerLesson/Extensions/AppTheme.swift の AppTheme.Palette に集約。個別 View で固定値は使用しない。

タイポグラフィ

SF Pro（system）のみ。.serif / .rounded / Cinzel は v1.1 で全廃。数値表示のみ .monospaced を許容。

役割サイズ目安weightdesign
大見出し（旧 displaySerif）22〜28pt.bold.default
セミ見出し（旧 titleSerif）16〜20pt.semibold.default
本文13〜15pt.regular / .medium.default
キャプション11〜12pt.regular / .medium.default
数値表示（XP・スタック・bb 等）11〜22pt.semibold / .bold.monospaced

大文字 tracking 強調は廃止（旧: tracking(4) 等の英字飾り）。

廃止した要素

  • shadow / drop shadow（全面廃止。境界は 1pt ボーダーで表現）
  • LinearGradient / RadialGradient / マルチストップグラデ（単色塗りへ統一）
  • backgroundVignette / tableFelt / heroSpotlight（CPU対戦・クイズテーブルの背景4層演出を全削除）
  • 角丸22以上（旧 cornerRadiusXL の 22 は 12 へ縮小）
  • tracking ≥ 3pt の英字飾り（display Serif と組み合わせていた装飾を排除）

コンポーネント

カード（goldCard ViewModifier）

呼び出し名は互換性のため goldCard() のまま。中身はフラットな1ptボーダーカードへ刷新済み。

// 旧：真鍮枠＋shadow
.goldCard()  // v1.0 では LinearGradient + shadow を含んでいた

// 新（v1.1）の実体
.background(RoundedRectangle(cornerRadius: 8).fill(AppTheme.Palette.surface))
.overlay(RoundedRectangle(cornerRadius: 8).strokeBorder(AppTheme.Palette.border, lineWidth: 1))

主CTA（primaryGoldButton ViewModifier）

アクセント色（#4C8DF6）の単色塗り。白文字、shadowなし。

Button { ... } label: { Text("レッスンを開始") }
  .primaryGoldButton()  // 中身は accent fill + Color.white foreground のフラット

セカンダリボタン（secondaryGoldButton ViewModifier）

surfaceElevated 背景 + 1pt ボーダー + textPrimary 文字。

選択chip（filter / picker 共通）

  • 未選択: surfaceElevated 背景 + border 1pt + textPrimary 文字
  • 選択中: accentSubtle 背景 + accent 1pt + accent 文字

アクションボタン（Quiz / CPU対戦の Raise / Call / Fold）

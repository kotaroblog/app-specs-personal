RoutineReel - ASOキーワード戦略

App Store Optimization の方針とキーワード設計（v1.0 リリース前）

ターゲットユーザー

  • ペルソナ20〜30代の SNS 発信意欲がある社会人・学生。TikTok/Reels 視聴者でルーティン動画を「見る」側から「投稿する」側に回りたい層
  • 課題続けたい朝/夜のルーティンが「撮影→編集→投稿」と分かれていて続かない。ステップを撮るだけで自動で1本にまとまる仕組みが欲しい
  • 利用シーン朝・夜の準備時間、ジム、勉強机。各ステップを5〜30秒で撮るだけで自動編集

App Store Connect 設定

  • アプリ名（30字）RoutineReel：ルーティン動画作成
  • サブタイトル（30字・A案）朝活・夜活・筋トレを動画日記に
  • サブタイトル（30字・B案）撮るだけで縦型ルーティン動画
  • サブタイトル（30字・C案）習慣を続けるための動画記録
  • プロモテキスト（170字）朝のルーティン、夜のスキンケア、筋トレ、勉強——毎日の習慣を5〜30秒ずつ撮るだけで、TikTok/Reels 風の縦型動画が自動で完成。テロップ・BGM・フィルタはワンタップ。続けて撮るだけ。
  • キーワードフィールド（100字）朝活,夜活,筋トレ,勉強,ルーティン,動画,編集,縦,リール,習慣,記録,タイマー,日課,継続,ショート,vlog,日記,スキンケア,動画日記,習慣化

説明文の構造（提案）

  1. 1行目（最重要）: 撮るだけでルーティン動画が完成。あなたの習慣を、毎日 1 分の動画に。
  2. 2〜3行目: 朝のスキンケアも、夜の整いも、筋トレも、勉強も——5〜30秒のステップを順に撮るだけで、TikTok / Reels 風の縦型動画がワンタップで完成します。
  3. 機能リスト:
    • ● 連続撮影 / クリップ蓄積の2モード（その場で全部撮る／好きな時間に貯める）
    • ● テロップ・BGM・フィルタを自動合成、編集不要
    • ● 朝5分 / 夜整い / 筋トレ10分 / 集中45分 のプリセット同梱
    • ● 撮影完了→TikTok / Reels に 1tap 投稿、ハッシュタグも自動コピー
    • ● 連続記録（streak）で習慣化サポート
  4. 使い方: ステップを設定 → 1ステップずつ撮影 → 自動編集 → TikTok / Reels へ投稿
  5. 差別化ポイント: 編集機能ではなく「続けやすさ」に全振り。撮るたびに自動編集 → 投稿まで最短経路。
  6. CTA: 今日のルーティンを、動画で残そう。

競合分析

競合アプリ強み弱み差別化ポイント
CapCut編集機能の豊富さ、テンプレ多数機能過多で習慣化できない、撮影フローが断絶「ルーティンを毎日撮る」体験そのものに特化
VLLO / InShot初心者向け編集撮影自体は別アプリ、ワークフロー断絶撮影→自動編集→投稿のワンストップ
Hevy / Strong筋トレ記録に特化動画化機能なし、SNS シェア弱い動画でルーティンを残せる、SNS バイラル設計
Stoic / Opal習慣化体験動画化機能なし、結果が目に見えない成果物（動画）が物理的に残る、SNS シェアで承認欲求も満たす
TikTok ネイティブ撮影編集との一体感毎日続けるための仕組みなしルーティン管理＋ streak で習慣化サポート

SNS 流入導線

  • 動画末尾アウトロカード: 完成動画のラストに QR コード（無料版のみ）と #RoutineReel を焼き込み。TikTok / Reels 視聴者の App Store 流入導線。
  • URL スキーム: routinereel://preset/<rawValue> で TikTok bio / Qiita / note から該当プリセット撮影画面へ 1tap 直送。
    プリセットURL
    朝5分ルーティンroutinereel://preset/morning
    夜の整いルーティンroutinereel://preset/night
    筋トレ10分ルーティンroutinereel://preset/workout
    集中45分ルーティンroutinereel://preset/study
  • Universal Link（リリース後検討）: https://kotaroblog.github.io/app-policies/routinereel/preset/<rawValue> を Universal Link 化し、未インストールユーザーをサポートページ→App Store へ誘導する。要 apple-app-site-association ファイル配置。
  • ハッシュタグ自動コピー: 完成動画のシェア時に #RoutineReel をクリップボードへ先回りコピー。TikTok/Reels 投稿画面で長押し→ペースト。

レビュー対策

  • ★4以上を維持する目標
  • レビュー依頼ダイアログの表示タイミング:
    • 動画保存5回目（既存 ReviewRequestService.onVideoSaved）
    • SNS シェア成功時（30日ガード付き・新規 onSharedToSNS）— 満足ピーク狙い
  • 低評価レビューへの返信方針: 機能要望は v1.1 以降に検討、バグ報告は最優先で修正

アニメーション原則

許可されたアニメーション

  • DesignsPressableButtonStyle: 押下時 scale 0.94 spring(0.28, 0.6)
  • DesignsFloatModifier: 浮遊（offset Y、永続）— 使用箇所限定
  • DesignsPulseModifier: 脈動（scale、永続）— REC indicator のみ
  • DesignsRingPulseModifier: 波紋（scale + opacity 片道）— Step progress 現在ステップのみ
  • easeOut フェード（appear時のみ）
  • spring 切替（タブ切替・ステップ切替）
  • blur transition（カウントダウン出現時のみ）

禁止アニメーション

  • loop animation（scale bounce 連続）
  • shimmer
  • floating UI（中央＋ボタンも v1.0 で削除）
  • bounce（spring overshoot 過多）
  • 常時 gradient shift
  • neon glow

共通コンポーネント

  • DesignsCardContainerpadding / cornerRadius / background / shadow を一括適用
  • designsSheetStyle()presentationCornerRadius 32 + presentationDragIndicator + presentationBackground
  • designsCardShadow()標準カード影を1行で適用

時間帯別ヘルパー（DesignsGreeting）

v1.0 では Hero モック削除のため未使用化。実データ算出機能（連続日数等）は v1.1 で復活予定。残置:

  • current: 時間帯別挨拶 + 絵文字（おはよう☀️ / こんにちは🌤 / お疲れさま🌙 / おかえり✨）
  • dateText: 日本語日付（M月d日 曜日）
  • moodCopy: 時間帯別軽コピー（今日もゆっくり始めよう / 少し整えよう / 5分だけ記録する？ / 静かに振り返ろう）

Home UI 統一原則

  1. 純黒禁止（背景は warm white、VideoPlayer 再生中のみ例外）
  2. 常時アニメーションを最小化（「Dribbble感」回避）
  3. glassmorphism を増やさない（既存 ultraThinMaterial / thinMaterial の範囲内）
  4. 余白を広く（DesignsSpacing.xl 以上を多用）
  5. カード cornerRadius は 16〜32 の範囲
  6. shadow opacity は 0.04〜0.12 の範囲
  7. 太字使いすぎ禁止（Hero以外は medium/regular）
  8. 「足すより止める」精神

動画合成仕様（VideoCompositionService）

完成動画の構成・フィルタ・テロップ・透かし・冒頭/末尾カードの全体仕様

実装ファイル

  • 合成本体Services/VideoCompositionService.swift
  • フィルタServices/VideoFilterService.swift + Models/VideoFilter.swift
  • タイムスタンプServices/AppSettings.swift（デフォルト非表示）

完成動画のレイヤー構成（時系列）

5レイヤーで構成。順番に重ねて焼き込まれる。

[0.0 ─── 0.6秒]  冒頭タイトルカード（全ユーザー・ログ風）
                  ↓
[0.6 ─── (合計尺-1.5)]  動画本体 + テロップ + 透かし(無料のみ)
                  ↓
[末尾 1.5秒]       アウトロカード（無料ユーザーのみ）

① 冒頭タイトルカード（v1.0で「ログ風」に刷新）

  • 尺0.6秒（0.40秒表示維持 → 0.20秒フェードアウト）
  • 背景上部28%のみグラデ（黒0.45 → 透明）。映像を遮らない
  • ルーティン名左上配置・動画高さ4.0%フォント・ユーザー選択フォント・opacity 0.95
  • 時刻ルーティン名の上に小さく（動画高さ2.4%・opacity 0.75・タイムスタンプON時のみ）
  • スキップ条件動画長 1.5秒未満なら表示しない

v1.0変更前: 黒全画面 opacity 0.55 + 中央配置 + ルーティン名6.0% → 評価で「広告っぽい」「テンプレ感」と指摘され、ログ風に刷新。

② 動画本体

  • 構成撮影クリップを順に AVMutableComposition で結合
  • アスペクト比9:16 縦型固定
  • 解像度ユーザー設定（720p / 1080p / 最高）
    低スペック機（4GB RAM 未満）は 1080p+ → 720p に自動降格
  • 音声撮影時の元音声のみ（BGMは v1.0 で完全廃止）
  • 元音声ミュートPreviewView の muteOriginalAudio トグルで無音化可能

③ テロップ焼き込み

v1.0で配置を 「中央 → 左下セーフゾーン」に変更（被写体回避 + TikTok/Reels UI衝突回避）。

  • ステップ名（メイン）左下配置・動画高さ3.4%フォント・x=5%、y=22%
  • 時刻（サブ）ステップ名の上に小さく配置・動画高さ2.4%フォント（ステップ名の70%）
  • アニメーション各クリップの時間範囲のみ opacity 1（CAKeyframeAnimation）
  • フォントユーザー選択（HelveticaNeue-Light等の6種）
  • 影黒シャドウで視認性確保

v1.0変更前: 右下に時刻を並列配置 → 右下のTikTok UI（いいね/コメント/シェア）と衝突 → 左下ステップ名の上に統合。

④ 透かし（v1.0で1行化・左上配置）

  • 対象無料ユーザーのみ・動画全体に常時表示
  • 位置左上（右下→左上に変更）
  • 内容「RoutineReel」のみ（CTA「App Storeで検索」は削除）
  • フォントHelveticaNeue-Medium・動画高さ2.0%・opacity 0.75
  • 影黒シャドウ radius 3

v1.0変更前: 右下・2行（ブランド+CTA）・opacity 0.85 → 「動画体験を削る」「右下UI衝突」と指摘され、左上1行に簡素化。

⑤ 末尾アウトロカード（v1.0で文言・尺・背景を改善・バイラル流入の核）

  • 対象無料ユーザーのみ
  • 尺1.5秒（旧 0.8秒）
  • スキップ条件動画長 3.0秒未満なら表示しない
  • 背景半透明黒 opacity 0.55（旧 0.70）
  • 上行「習慣を、動画で残す。」（小・opacity 0.85）— 旧「Shot with」（英語混在）から日本語の余韻文言へ
  • 中央「RoutineReel」（動画高さ7.0%・bold・特大）
  • 下行「App Store で検索」（動画高さ2.2%・opacity 0.78）
  • アニメーション最初の0.15秒でフェードイン → 最後まで表示

フィルタ8種（v1.0で「セピア」削除→「Clean」追加）

無料（3種）

ID表示名効果
noneなし無加工
grain粒子フィルム粒子を薄く重ねる（CIRandomGenerator intensity 0.20）
cleanクリーンv1.0新規: 彩度+5% / 明度+3% / コントラスト+2% / シャドウ持ち上げ（CIHighlightShadowAdjust shadowAmount 0.3）

Pro（5種）

ID表示名効果
film8mm8mm暖色（CITemperatureAndTint 7500K）+ 粒子0.15 + ビネット（CIVignette radius 2.5）
vhsVHS彩度85% + コントラスト105% + 色相シフト5% + 粒子0.25
polaroidポラロイド彩度85% + 明度+5% + 暖色（7200K）
noirモノクロCIPhotoEffectNoir（白黒）
faded色褪せ彩度70% + コントラスト90% + シャドウ持ち上げ（90sフィルム風）

v1.0変更: 「セピア」を削除し「Clean」を追加。20代女性vlogger層（モーニングルーティン/スキンケア）の使用率が低いセピアを排除し、朝活/勉強vlog向けナチュラル整えフィルタへ置換。Claude.ai/GPT 両AI評価で一致した削除推奨。

タイムスタンプ表示

  • 設定箇所AppSettings.timestampFormat（24時間制 / 12時間制 / 非表示）
  • v1.0デフォルト非表示（旧 24時間制）
  • 変更理由投稿事故防止（夜更かしや週末バレを意図せず映す事故）
  • 表示有効時冒頭カードの時刻 + テロップ内の時刻 が表示される

BGM の扱い（v1.0で完全廃止）

v1.0 で完全削除。元音声のみ。詳細は BGM機能削除の意思決定記録。

動画例（5ステップ×8秒のモーニングルーティン）

[0.0 - 0.6秒]  左上「7:30」+「平日モーニングルーティン」（上部グラデのみ）
[0.6 - 8.6秒] 起床・水を飲む（左下時刻+ステップ名 + 左上小「RoutineReel」）
[8.6 - 16.6秒] スキンケア
[16.6 - 24.6秒] 着替え
[24.6 - 32.6秒] コーヒー
[32.6 - 40.6秒] 外出
[39.1 - 40.6秒] 「習慣を、動画で残す。RoutineReel / App Store で検索」（黒0.55・無料のみ）

合計 約 40.6秒

合成パイプライン

  1. クリップURL検証（FileManager 存在チェック）
  2. AVMutableComposition への各クリップ video/audio トラック挿入
  3. 各クリップの時間範囲を記録（テロップ表示制御用）
  4. renderSize 決定（最初のクリップから）
  5. AVMutableVideoComposition + AVVideoCompositionInstruction で transform 適用
  6. parentLayer に各レイヤー追加: テロップ → 透かし → 冒頭カード → アウトロカード
  7. 元音声ミュート設定（必要時）
  8. AVAssetExportSession で書き出し（プリセット自動選択）
  9. 進捗ポーリング（0.1秒間隔、progressHandler 経由）
  10. メモリ警告監視（書き出し中・OOM防止）

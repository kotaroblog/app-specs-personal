設定画面（SettingsView）

Card section 化により List/Form 感を完全除去。Stoic/Opal 系の静かな設定画面。

基本情報

  • 画面IDsettings
  • 実装ファイルViews/Settings/SettingsView.swift
  • 表示方式sheet（モーダル）
  • 起動経路ホーム上部歯車アイコン / 下部設定タブ
  • 背景色#FAF9F7

セクション構成

  1. プラン: RoutineReel Pro（Pro時:「ご利用ありがとうございます」/ 無料時:「記録をもっと自由に」）
  2. 一般: デフォルト撮影秒数 / 動画画質 / タイムスタンプ形式
  3. β機能: 新しいホーム画面を使う（FeatureFlag切替）
  4. 他のアプリ: WorkoutDiary / StudyStopwatch（App Store 誘導）
  5. 情報: プライバシーポリシー / 利用規約 / サポート / レビューを書く
  6. デバッグ: Pro機能を有効化（#if DEBUG || BETA のみ表示）

Card section 化の原則

  • レイアウトScrollView + VStack（List/Form 不使用）
  • セクションヘッダーuppercase + tracking 0.4 + tertiary text（控えめ）
  • カード背景Color.designsCard + designsCardShadow
  • カード角丸DesignsRadius.lg (20pt)
  • row高さ最小58pt（呼吸感確保）
  • アイコンSF Symbols・accent orange opacity 0.7（pastel tint）
  • subtitle opacity0.85（メインテキストより薄く）
  • Divider左paddingでアイコン分インデント（24pt + spacing.md）

β機能セクション（最重要）

FeatureFlag useDesignHome の切替トグル。リリースビルドでも常時表示。

  • タイトル新しいホーム画面を使う
  • subtitle不具合があればオフにしてください
  • footerホーム画面のデザインを試験的に切り替えられます。
  • アイコンsparkle（accent orange）
  • 永続化@AppStorage(FeatureFlags.useDesignHomeKey) Bool = true
  • 即時反映トグル変更後シートを閉じると RootView が再評価され、次回起動時に切替

→ 詳細は FeatureFlag運用 参照。

Pro セクション設計

  • "Upgrade Now!"禁止: アグレッシブな課金誘導は使わない
  • Pro 時: 静的カード「RoutineReel Pro / ご利用ありがとうございます」
  • 無料時: Button カード「RoutineReel Pro / 記録をもっと自由に」+ chevron right
  • アイコンは crown.fill + soft warm gradient 円背景

設定画面（SettingsView）

Card section 化により List/Form 感を完全除去。Stoic/Opal 系の静かな設定画面。

基本情報

  • 画面IDsettings
  • 実装ファイルViews/Settings/SettingsView.swift
  • 表示方式sheet（モーダル）
  • 起動経路ホーム上部歯車アイコン / 下部設定タブ
  • 設定タブアイコンgearshape.fill（旧: person.crop.circle、596896d で変更）
  • 背景色#FAF9F7

セクション構成

  1. プラン: RoutineReel Pro（Pro時:「ご利用ありがとうございます」/ 無料時:「記録をもっと自由に」）
  2. 一般: デフォルト撮影秒数 / 動画画質 / タイムスタンプ形式
  3. 他のアプリ: WorkoutDiary / StudyStopwatch（App Store 検索リンク）
  4. 情報: プライバシーポリシー / 利用規約 / サポート / レビューを書く（596896d で実装）
  5. デバッグ: Pro機能を有効化（#if DEBUG || BETA のみ表示）

β機能セクション（削除済み・v1.2）: 「新しいホーム画面を使う」FeatureFlag 切替トグルは FeatureFlags.swift 廃止にともない設定画面から削除した。

Card section 化の原則

  • レイアウトScrollView + VStack（List/Form 不使用）
  • セクションヘッダーuppercase + tracking 0.4 + tertiary text（控えめ）
  • カード背景Color.designsCard + designsCardShadow
  • カード角丸DesignsRadius.lg (20pt)
  • row高さ最小58pt（呼吸感確保）
  • アイコンSF Symbols・accent orange opacity 0.7（pastel tint）
  • subtitle opacity0.85（メインテキストより薄く）
  • Divider左paddingでアイコン分インデント（24pt + spacing.md）

β機能セクション（廃止済み・v1.2）

FeatureFlags.swift の削除にともない、SettingsView から「β機能」セクション（「新しいホーム画面を使う」トグル）を削除した。DesignsHomeView が唯一のホーム画面として固定されたため、実行時の UI 切り替えは不可能（意図的）。詳細は FeatureFlag運用（廃止記録） を参照。

Pro セクション設計

  • "Upgrade Now!"禁止: アグレッシブな課金誘導は使わない
  • Pro 時: 静的カード「RoutineReel Pro / ご利用ありがとうございます」
  • 無料時: Button カード「RoutineReel Pro / アプリ内保存 X/3 ・記録をもっと自由に」+ chevron right
  • アイコンは crown.fill + soft warm gradient 円背景

ファイル構成（v1.2 で分割）

SettingsView.swift から SettingsLegalDocument / SettingsLegalDocumentView / SettingsSupportView を SettingsLegalViews.swift に分離した（コード整理のみ・UX への影響なし）。

  • SettingsView.swiftメイン設定画面（セクション一覧 / プラン / 一般 / 他アプリ / 情報 / デバッグ）
  • SettingsLegalViews.swiftSettingsLegalDocument（enum）/ SettingsLegalDocumentView（WebView wrapper）/ SettingsSupportView を格納

保存メーター統合（v1.0.3 / 2026-06-10）

Free ユーザー向け Pro アップグレードカード（proUpgradeCard）に、アプリ内保存件数の現在値を副題内に統合表示する。Paywall への自然な動機付けとして機能。

  • データソース@Query private var allVideos: [Video] を SettingsView に追加
  • 分子allVideos.count（アプリ内保存件数）
  • 分母VideoStorageService.freeStorageLimit = 3
  • 副題テキスト「アプリ内保存 X/3 ・記録をもっと自由に」
  • 上限到達時の強調X が 3 に達した場合は text 色をピンクに切替（注意喚起）
  • accessibilityLabel「RoutineReel Pro にアップグレード。アプリ内保存 X 件中 3 件上限」
  • Pro ユーザー非表示（既存「ご利用ありがとうございます」カードのまま）
  • 関連DesignsHomeView の「最近の動画」見出し横 storageMeterBadge と整合（Free: X/3、Pro: ∞）

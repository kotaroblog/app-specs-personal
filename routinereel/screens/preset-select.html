プリセット選択（PresetSelectView）

新規ルーティン作成時の入口。プリセットから始めるかカスタムを選ぶ。v1.0で Home UI 統一 + 日本語化。

基本情報

  • 画面IDpreset-select
  • 実装ファイルViews/Home/PresetSelectView.swift
  • 表示方式sheet（モーダル）
  • 起動経路中央＋ボタン / 新規ルーティン作成ボタン
  • 背景色#FAF9F7 (warm white)

画面構成

  1. タイトル: 「ルーティンを選ぶ」（26pt bold）+ サブコピー「プリセットから始めるか、自分で作成できます。」
  2. プリセット4種: morning / night / workout / study（カード形式）
  3. 自分で作成: ダッシュ枠 + accent orange の独立ボタン

プリセットカード

絵文字名前ステップ
☀️モーニングルーティンwake up → skincare → outfit → coffee → go out
🌙ナイトルーティンback home → bath → skincare → reading → sleep
💪ワークアウトwarm up → training → stretch → protein → shower
📚勉強ルーティンprep → study → break → study → review

カード仕様

  • 背景Color.designsCard + designsCardShadow
  • cornerRadiusDesignsRadius.lg (20pt)
  • 構成絵文字22pt + 名前16pt semibold + ステップ数 + chevron
  • サブテキストステップ名を「→」連結（11pt light tertiary、2行まで）
  • 押下DesignsPressableButtonStyle scale 0.98

「自分で作成」ボタン

  • 背景accent orange opacity 0.08 + ダッシュ枠 lineWidth 1 dash [4]
  • アイコンpencil
  • 文言「自分で作成」(15pt semibold accent orange)
  • 挙動selectedPreset = nil → showEditView = true で RoutineEditView(preset: nil) 起動

選択フロー

1. ユーザーがプリセット選択（または自分で作成）
2. createFromPreset(preset) → selectedPreset = preset
3. showEditView = true で .sheet 起動
4. RoutineEditView(routine: nil, preset: selectedPreset) を表示
5. RoutineEditView.loadData() で preset.displayName と preset.steps を初期値にセット
6. ユーザーが編集して保存 → 新規 Routine 作成

RoutinePreset と統合（v1.0で日本語化）

各プリセット情報は Models/RoutinePreset.swift に集約。PresetSelectView 内でローカルヘルパー（presetEmoji / presetTitleJa）を持っていたが、コードレビューで重複指摘 → 削除し RoutinePreset のプロパティを直接参照する形に整理。

// 旧: PresetSelectView 内のローカル関数
private func presetEmoji(_ preset: RoutinePreset) -> String { ... }
private func presetTitleJa(_ preset: RoutinePreset) -> String { ... }

// 新: RoutinePreset 直下のプロパティ
preset.emoji         // "☀️"
preset.displayName   // "モーニングルーティン"

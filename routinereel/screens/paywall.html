ペイウォール（PaywallView）

Home UI 統一版。warm white 背景 + pink→orange CTA で「習慣記録プロダクト」の世界観を維持。

基本情報

  • 画面IDpaywall
  • 実装ファイルViews/Common/PaywallView.swift
  • 表示方式sheet（モーダル）
  • 起動経路①Pro バッジ ②課金壁発火（ルーティン3件目・複製・Quick Start）③設定画面のPro導線
  • 背景色#FAF9F7（純黒禁止）

画面構成

  • ①閉じるボタン右上 30pt 円・opacity 0.6・存在感弱め
  • ②Heroメインコピー「毎日の記録を、もっと心地よく。」+ サブコピー「透かしなし・無制限保存・広告非表示」
  • ③特典リスト4項目・SF Symbols（infinity / sparkles / wand.and.stars / square.and.arrow.up）+ soft warm gradient 円背景
  • ④プラン選択年額（おすすめ・pink→orange グラデバッジ）+ 月額。Card セレクトUI
  • ⑤購入CTA「7日間無料で試す」/「月額プランで始める」 + 補足「いつでもキャンセル可能」
  • ⑥フッター「あとで」「購入を復元」（小さく控えめ）

Pro 特典（4項目）

アイコンタイトルサブテキスト
infinityルーティン無制限撮りたいだけ作成できます
sparkles透かしなしの書き出しクリーンな動画として保存
wand.and.stars全フィルタ・全フォント解放雰囲気を自由に演出
square.and.arrow.up広告非表示集中して記録できる

※BGM特典は v1.0 で完全削除（BGM機能削除の意思決定記録 参照）

プラン設計

  • 月額¥480（com.happyboy1002.RoutineReel.monthly）
  • 年額¥3,800・7日間無料トライアル付（com.happyboy1002.RoutineReel.yearly）
  • 月換算（年額）¥317（34%オフ訴求）
  • デフォルト選択年額（おすすめバッジ付き）

CTA仕様（Home CTA と統一）

  • 高さ58pt
  • cornerRadius28pt
  • 背景LinearGradient（pink → orange）+ 右上 RadialGradient 白30%（生活感）
  • shadowaccent pink 35%・radius 14・y 6
  • 押下scale 0.97 spring（DesignsPressableButtonStyle）
  • 無効化subscription.isLoading || productToPurchase == nil

アクセシビリティ

  • 閉じるボタン: accessibilityLabel("閉じる")
  • プラン選択: "年額プラン ¥3,800、おすすめ" 形式の読み上げ + .isSelected trait
  • 購入CTA: accessibilityLabel(purchaseButtonLabel) + accessibilityHint("いつでもキャンセル可能")
  • あとで / 購入を復元: それぞれ専用ラベル

Home UI 統一原則の遵守

  • ✅ 純黒禁止 → warm white
  • ✅ 常時animation禁止 → PressableButtonStyle のみ
  • ✅ glassmorphism 増やさず → 既存維持
  • ✅ 余白を広く → DesignsSpacing.xxl 多用
  • ✅ shadow opacity 0.08〜0.12 → DesignsShadowStyle で集約
  • ✅ 太字使いすぎ禁止 → Hero以外は medium/regular

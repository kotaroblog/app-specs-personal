アップデートお知らせ機能（What's New）

新バージョン起動時にリリースノートをシート表示し、更新内容をユーザーに通知する

v1.0 機能ID: WD-FEAT-RELEASENOTES

概要

アプリ更新後の初回起動時に「What's New」シートを自動表示し、バージョンごとのリリースノートをユーザーに伝える機能。オンボーディングと重複しないよう制御され、設定画面からもいつでも閲覧できる。リリースノートはアプリ内ハードコードで管理する。

関連画面

  • 設定画面（WD-SETTINGS-01） — 「App Info」セクションに「What's New」エントリを常設

モデル仕様

プロパティ型説明
versionStringバージョン文字列（例: "1.0.7"）
dateStringリリース日（例: "2026-05-14"）
highlights[ReleaseHighlight]変更点の配列
ReleaseHighlight プロパティ型説明
iconStringSF Symbols アイコン名
titleKeyStringローカライズキー（en/ja対応）
descriptionKeyStringローカライズキー（en/ja対応）

処理フロー

sequenceDiagram participant App as ContentView participant Svc as ReleaseNotesService participant UD as UserDefaults participant UI as WhatsNewView App->>Svc: checkShouldShowWhatsNew() Svc->>UD: get("lastSeenReleaseVersion") UD-->>Svc: 前回バージョン or nil Svc->>Svc: 現在バージョンと比較 alt 新バージョン かつ 初回起動でない Svc-->>App: shouldShow = true App->>UI: sheet 表示 UI-->>App: 閉じる App->>Svc: markAsSeen(version:) Svc->>UD: set("lastSeenReleaseVersion", 現在バージョン) else 初回起動 or 同バージョン Svc-->>App: shouldShow = false end

ビジネスルール

  • UserDefaults キー lastSeenReleaseVersion で前回起動バージョンを管理する
  • 現在バージョンが lastSeenReleaseVersion より大きい場合のみシートを表示する
  • lastSeenReleaseVersion が存在しない（初回起動）場合はシートを表示しない。オンボーディングを優先させるため
  • 表示後に lastSeenReleaseVersion を現在バージョンに更新する
  • 設定画面「App Info」セクションの「What's New」エントリは常時表示。タップで WhatsNewView をシート表示する
  • リリースノートは en / ja 両対応（Localizable.xcstrings によるローカライズ）
  • リリースノートのデータはコードにハードコード（ReleaseNotesService 内配列）で管理する。サーバー取得は行わない

外部連携

連携先用途備考
UserDefaultslastSeenReleaseVersion の読み書きKeychain 不要（機密情報なし）

エラー処理

発生条件対応
バージョン文字列が取得できない（Bundle.main.infoDictionary 異常）シート表示をスキップし、Logger で warning を出力

実装メモ

  • ReleaseNote.swift — データモデル（Models/）
  • ReleaseNotesService.swift — バージョン比較・UserDefaults 管理（Services/）
  • WhatsNewView.swift — シート表示UI（Views/Settings/）
  • ContentView.swift — アプリ起動時に checkShouldShowWhatsNew() を呼び出し
  • SettingsView.swift — 「App Info」セクションに「What's New」エントリ追加
  • en/ja ローカライズ対応（Localizable.xcstrings）
  • 将来的にサーバーから動的取得する場合は ReleaseNotesService のデータソースのみ変更（インターフェースは変えない）

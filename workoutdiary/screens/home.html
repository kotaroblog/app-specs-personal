ホーム画面

トレーニング記録の一覧表示・新規記録の起点

v1.0 画面ID: HOME-01

目的・役割

ユーザーが過去のトレーニング記録を時系列で確認し、新規記録の追加へ素早く遷移するための画面。 アプリ起動直後の主要なエントリポイント。

UI 構成

要素種類説明
ナビゲーションバー固定タイトル「ホーム」、右上にカレンダーアイコン
記録リストList日付別にグルーピング、最新が上
FAB（＋ボタン）フローティング右下、新規記録画面を起動
空状態EmptyView記録ゼロ時に表示

状態

状態表示内容
初期表示ローディングインジケータ
記録あり日付別グループのリスト
記録なし「最初のワークアウトを記録しよう」CTA

遷移

操作遷移先
FAB タップ新規記録画面
リストセル タップワークアウト詳細
カレンダーアイコン タップカレンダー画面
左スワイプ削除確認アラート

使用するデータモデル

@Model
final class Workout {
    var id: UUID
    var date: Date
    var exercises: [Exercise]
    var note: String?
    var createdAt: Date

    init(date: Date = .now) {
        self.id = UUID()
        self.date = date
        self.exercises = []
        self.createdAt = .now
    }
}

実装メモ

  • SwiftData の @Query で日付降順取得
  • セクションヘッダーで日付グルーピング
  • 無料ユーザーには30日以前の記録閲覧時に Paywall 表示
  • 引き下げリフレッシュで最新化

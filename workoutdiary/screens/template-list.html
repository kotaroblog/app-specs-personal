ルーティン選択（TemplateListView）

保存済みのルーティン（ワークアウトテンプレート）を一覧・適用・編集する

v1.1画面ID: WD-TEMPLATE-01

目的・役割

頻繁に行うメニュー（例: 胸の日 / 脚の日 / プッシュデー）をルーティンとして保存・適用する画面。v1.1.0 でラベルを「テンプレート」から「ルーティン」へ変更し、曜日バッジ表示・「➕/✏️/🗑」アイコン常時表示を追加した。内部モデル名 WorkoutTemplate は互換維持のため変更なし。

UI 構成（v1.1.0 以降）

要素種類説明
ルーティンリストList（insetGrouped）ルーティン名・曜日バッジ・エントリプレビュー（種目名・重量×回数×セット数）を表示
曜日バッジCapsule TextassignedWeekdays が空でない場合、ルーティン名横に「月・水・金」形式で表示（AppTheme.accent）
➕（ワークアウト追加）アイコンButton（各行右端）タップで loadTemplate() を実行し dismiss。行タップと同機能
✏️（編集）アイコンButton（各行右端）EditTemplateView を sheet 表示
🗑（削除）アイコンButton（各行右端）削除確認ダイアログを表示後に削除。確認なし即削除ではない
エンプティ表示ContentUnavailableView 相当テンプレートが0件の場合「No Templates」を表示
v1.1.0 変更点
- UIラベル「テンプレート」→「ルーティン」に全置換（navigationTitle / ボタンラベル等）
- スワイプアクション削除を廃止し、「➕・✏️・🗑」アイコンを各行右端に常時表示
- assignedWeekdays をもとに曜日バッジを表示
- 行タップ / ➕アイコンタップ = ルーティン適用して dismiss（確認ダイアログ廃止）
内部モデル名（WorkoutTemplate）・localstoreキー等は変更なし（SwiftData互換維持）

遷移

操作遷移先
行タップ / ➕タップloadTemplate() 実行 → dismiss（WorkoutSession に追加して閉じる）
✏️タップEditTemplateView (sheet)
🗑タップ削除確認ダイアログ → 削除 or キャンセル

使用するデータモデル

@Model final class WorkoutTemplate {
    var id: UUID
    var name: String
    var createdAt: Date
    // v1.1.0 追加：曜日固定ルーティン
    var assignedWeekdays: [Int] = []   // 0=日, 1=月, ..., 6=土。空=曜日固定なし
    @Relationship(deleteRule: .cascade, inverse: \TemplateEntry.template)
    var entries: [TemplateEntry] = []

    var sortedEntries: [TemplateEntry] { ... }
    func appliesTo(date: Date) -> Bool  // 曜日判定
}

@Model final class TemplateEntry {
    var exerciseName: String
    var exerciseCategory: String
    var weight: Double
    var reps: Int
    var sets: Int
    var order: Int
    var duration: Double   // 有酸素: 分
    var distance: Double   // 有酸素: km
    var isCardio: Bool = false
}

実装メモ

  • 適用時に WorkoutSet を entry.sets 数だけ INSERT（safeSets = min(entry.sets, 20) で上限制御）
  • 経験・目標で重量・レップを自動補正（UserProfile.goal / UserProfile.experience 参照）
  • 初回起動時に insertDefaultTemplates() でデフォルト挿入
  • v1.1.0: スワイプ削除を廃止し「➕・✏️・🗑」アイコン常時表示に変更
  • v1.1.0: 曜日バッジ weekdayBadge(_ weekdays:) で「月・水・金」形式に変換
  • v1.1.0: 行タップ = ➕アイコンタップ（loadTemplate → dismiss）に統一
  • TemplateListView は WorkoutSessionView から sheet 表示（date: Date を引数で受け取る）

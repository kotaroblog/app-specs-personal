ホーム画面ウィジェット（3種）

Summary / Weekly / CategoryTracker

v1.0.5画面ID: WD-WIDGET-01〜03

目的・役割

ホーム画面に配置できるウィジェット3種。週間進捗・継続日数・カテゴリ別トラッキングを表示。

3種類のウィジェット

ウィジェットサイズ表示内容
SummarySmall今週のセッション数・最終トレーニング日・ストリーク
WeeklyMedium週間サマリ + トップ種目
CategoryTrackerMediumカテゴリ別の今週セッション数

データソース

// AppGroup UserDefaults
WidgetSharedData {
    let weeklyCount: Int
    let lastWorkoutDate: Date?
    let currentStreak: Int
}

WeeklySummaryData {
    let weekStart: Date
    let weekEnd: Date
    let totalSessionCount: Int
    let topExercises: [TopExercise]
    let currentStreak: Int
    let streakUnit: StreakUnit  // .days / .weeks
}

更新タイミング

  • 本体起動時に WidgetSharedData を AppGroup に書き込み
  • セッション保存後に WidgetCenter.shared.reloadAllTimelines()
  • 翌日0時に自動再取得

実装メモ

  • Bundle Identifier: com.happyboy1002.WorkoutDiary.widget
  • AppGroup: 共通の suiteName で本体・Widget が共有

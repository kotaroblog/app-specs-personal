インターバルタイマー（IntervalTimerView）

セット間休憩のカウントダウンタイマー

v1.0画面ID: WD-TIMER-01

目的・役割

セット間の休憩時間を計測するタイマー画面。バックグラウンド継続対応。

UI 構成

要素種類説明
残り時間表示大きなTextMM:SS 形式
時間プリセットHStack30s / 60s / 90s / 120s / 180s
カスタム入力Stepper任意秒指定
開始/一時停止/リセットButtonタイマー制御
サウンド設定Toggle終了音の有無

状態

状態表示内容
停止中「開始」ボタン
カウント中「一時停止」ボタン + 残り秒数
終了サウンド再生 + リセットボタン

実装メモ

  • IntervalTimerManager で Timer 制御
  • scenePhase 監視でバックグラウンド時も継続
  • 終了時にハプティクス + サウンド

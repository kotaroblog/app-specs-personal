ゲームメイン画面

SceneKit 3D フィールド上に SwiftUI HUD をオーバーレイするコアプレイ画面。サバイバル / クリエイティブの両モードをカバーする。

画面ID: CF-S01

目的・役割

ゲームの中核となる画面。SCNView（SceneKit）がフルスクリーンでレンダリングを担い、その上に SwiftUI の ZStack で HUD・操作コントロールをオーバーレイする構成をとる。 サバイバルモードとクリエイティブモードで表示・操作が一部切り替わる。各種パネル（インベントリ・クラフト・かまど・チェスト・設定・死亡）もこの画面上にシート / オーバーレイで展開される。

クロスヘア（中央）

画面中央に常時固定表示。照準先にブロックが存在する場合はブロック外周をハイライト。照準先なし（空中・遠距離）の場合はハイライトなし。

ホットバー（下中央）

選択中ブロック種別のアイコンと所持個数バッジを表示するスロット行。スロットをタップして配置ブロックを切り替える。クリエイティブモードでは個数表示なし（無限）。

死亡・復活フロー

ステップ処理
1. HP が 0 になる死亡原因を記録（溶岩落下 / 落下ダメージ / モブ攻撃 等）
2. 赤フェードSCNView に赤いオーバーレイをフェードイン
3. 死亡オーバーレイ表示「あなたは死にました」+ 死亡原因テキスト + 「復活する」「終了して戻る」ボタン
4a. 「復活する」最終スポーン地点（最後に寝たベッド or ワールド初期スポーン）にリスポーン。HP / 空腹を回復して再開
4b. 「終了して戻る」最終プレイ日時を更新してワールド選択画面へ pop

使用するデータモデル

// プレイヤー状態（非永続・インメモリ）
struct Player {
    var position: SCNVector3
    var velocity: SCNVector3
    var hp: Int           // 最大 20
    var hunger: Float     // 最大 20
    var yaw: Float
    var pitch: Float
    var isFlying: Bool
    var inventory: [BlockType: Int]
    var unlockedTools: Set<ToolType>
    var selectedTool: ToolType
    var selectedBlock: BlockType
}

// ToolType enum（採掘速度倍率を保持）
enum ToolType: String {
    case fist, woodPickaxe, stonePickaxe, ironPickaxe, diamondPickaxe

    var miningSpeedMultiplier: Float { /* 1, 2, 4, 8, 16 */ }
    var craftCost: (material: BlockType, count: Int)? { /* nil = 初期解放済み */ }
}

// UI・ゲーム状態（@Observable @MainActor）
@Observable @MainActor final class GameState {
    var isCreativeMode: Bool = false
    var currentWeather: WeatherType = .clear
    var graphicsQuality: GraphicsQuality = .standard
    var showCraft: Bool = false
    var showInventory: Bool = false
    var showFurnace: Bool = false
    var showChest: Bool = false
    var showSettings: Bool = false
    var isPlayerDead: Bool = false
    var deathCause: String = ""
}

// SwiftData 永続モデル
@Model
final class WorldRecord {
    var id: UUID = UUID()
    var name: String = ""
    var seed: Int = 0
    var lastPlayedAt: Date = Date()
}

実装メモ

  • SCNView を UIViewRepresentable でラップし、SwiftUI の ZStack 最背面に配置。HUD は ZStack の前面レイヤーに重ねる
  • ゲームループは SCNSceneRendererDelegate の renderer(_:updateAtTime:) で管理（目標 60fps）
  • パネル表示中は scnView.isPaused = true でレンダリングを停止しバッテリー節約
  • ジョイスティックと右エリア DragGesture は SimultaneousGesture で並列処理し、multiTouch を有効化する
  • 視点感度・デッドゾーンは UserDefaults に保存し、GameState 経由でジョイスティックロジックに反映する
  • 壊すボタンの長押し判定は DragGesture(minimumDistance: 0) の onChanged / onEnded で実装し、採掘進行度を Timer で更新する
  • 採掘完了時はツールの miningSpeedMultiplier を基準時間に掛けて採掘時間を算出する
  • 亀裂エフェクトは SCNMaterial のテクスチャを採掘進行度（0.0〜1.0）に応じて段階的に差し替えて表現する（6 段階テクスチャ）
  • クリエイティブモードへの切替はゲーム中いつでも可能だが、サバイバル → クリエイティブに切り替えた場合のドロップアイテムや経験値は消去しない
  • 死亡フェードは SCNAction.sequence で赤オーバーレイ SCNNode をフェードイン後、GameState.isPlayerDead = true で SwiftUI オーバーレイを表示する
  • 設定パネルの画質プリセットは SCNView の antialiasingMode / shadowMapSize 等を切り替えて反映する
  • 横画面固定は Info.plist の UIInterfaceOrientationLandscapeLeft / UIInterfaceOrientationLandscapeRight を設定し、SwiftUI の .supportedOrientations でも制限する
